# PII Breach 事件 — 法務記錄

## §1 事件 classification

依個資法 §12 + 施行細則 §22，本事件構成個人資料外洩。需即時通報主管機關（PDPC 籌備處）並通知當事人。

## §時間軸

| 時間 (ISO 8601) | 事件 | 來源 |
|---|---|---|
| 2026-05-10 | 異常存取偵測 | SIEM log |
| 2026-05-11 | 內部確認外洩 | IT incident review |
| 2026-05-13 | 法務知悉 | 通報流程 |
| ⏳ 待通報 | PDPC 通報日 (即時) | 施行細則 §22 |
| ⏳ 待通知 | 當事人通知日 | 個資法 §12 |

## §2 法源依據

- 個資法 §12 通知義務（公務+非公務）
- 個資法施行細則 §22 即時通報 PDPC
- 個資法 §27 安全維護義務（事後檢討）
