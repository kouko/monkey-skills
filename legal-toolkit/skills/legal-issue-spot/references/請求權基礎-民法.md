## 民法 §184 (侵權行為)

一般侵權行為的 frame——加害人因 故意/過失 不法侵害 他人權利，造成
損害 且具 相當因果關係 時，負損害賠償責任。第 II 項另開 違反保護他人
法律 之 推定過失 路徑 (學說：第 I 項前段「權利」、後段「故意以背於
善良風俗加損害」、第 II 項「違反保護他人法律」三 hypotheses)。

| 構成要件 element | 1-line 白話 | 典型 反例 | 典型 carve-out |
|---|---|---|---|
| 行為 (作為或不作為) | 須有可歸責之 加害行為，包含 不作為 (法定義務存在時) | 不可抗力之自然事件 (天災)；無法定/契約 作為義務之 純然旁觀 | §184 II 違反保護他人法律 → 不作為 推定為 加害行為 |
| 故意或過失 | 行為人主觀有預見可能 (故意) 或未盡 善良管理人 注意義務 (過失) | 已盡 善良管理人 注意仍無法避免；純物理意外、不可抗力 | §184 II 違反保護他人法律 → 推定過失，舉證責任倒置；§191-191-3 商品/動物/危險工作 推定過失 |
| 不法 | 違反法定義務 / 保護他人法律 / 善良風俗 | §149 正當防衛；§150 緊急避難；§151 自助行為；正當業務行為；被害人有效同意 | 業務上行為超過必要程度；同意範圍以外之行為仍為不法 |
| 侵害 他人之 權利 (§184 I 前段) | 侵害 受保護之 絕對權 (人格權 / 身分權 / 物權 / 智慧財產權) | 純粹經濟損失 (pure economic loss) — 通說：§184 I 前段 不及；契約債權之被侵害 (學說：第三人侵害債權 仍有爭議) | 走 §184 I 後段「故意以背於善良風俗」或 §184 II「違反保護他人法律」路徑 (學說：個資外洩 是否構成「權利」侵害有 人格權說 vs 隱私權說 對立) |
| 損害 之發生 | 實際發生且可量化之 財產上 / 非財產上 不利益 | 純粹擔憂 / 主觀不適 但無客觀損害；尚未實現之 將來假設利益 | §195 人格權侵害之 慰撫金 (非財產上損害)；§216 所失利益 (預期利益) |
| 相當因果關係 | 條件因果 + 適當性 (依社會經驗該行為通常會生此損害) | 介入行為 (intervening cause) 切斷因果；損害肇因於 第三人 獨立行為；被害人 重大過失 自陷風險 | §217 過失相抵 (減輕但不免除)；§185 共同侵權行為人 連帶責任，因果不切斷 |
| (§184 II 路徑) 違反 保護他人 之 法律 | 行為違反以保護個人權益為目的之 法律 / 命令 (e.g. 個資法 §27 安措 / 勞基法 §59 職災補償) | 行政管制法令 但 立法目的 非保護個人 (e.g. 純粹秩序罰)；雖違反法律但 損害類型 不在法律保護範圍內 | 推定過失：行為人須舉證 已盡相當注意；保護目的 解釋 — 若法令同時保護個人，仍適用 (學說：保護他人法律 範圍 通說 vs 限縮說) |

**§184 路徑速記**：

- §184 I 前段「權利」：限 絕對權；個資 / 純粹經濟損失 走後段或 II 項
- §184 I 後段「背於善良風俗」：故意 為要件，門檻較高
- §184 II「違反保護他人法律」：推定過失 + 不限「權利」，常見於個資 / 勞動 / 食安

## 民法 §227 (不完全給付)

債務人雖有 給付，但 給付內容 不符 債之本旨 (瑕疵給付) 或因 給付行為
另造成 債權人 其他損害 (加害給付) 時的責任。第 II 項 加害給付 與 §184
侵權行為 可能 請求權競合 (學說：§227 II 加害給付 vs §184 II 違反保護
他人法律 — 通說採 請求權競合，債權人 擇一行使)。

| 構成要件 element | 1-line 白話 | 典型 反例 | 典型 carve-out |
|---|---|---|---|
| 債之關係 存在 | 須先存在有效 契約 / 法定 之債之關係 | 契約不成立 / 無效 / 撤銷後失效；純粹 事實上 接觸 (走 §184 不走 §227) | §245-1 締約上過失 (契約成立前的信賴利益保護) |
| 可歸責於 債務人 之事由 | 債務人具 故意 或 過失 (§220 I：除契約另有訂定，債務人 過失即負責) | 不可抗力 (§225 I 給付不能之 風險分配)；債權人 受領遲延 中之事故 (§237) | §220 II 故意/重大過失 不得預先免除；債務人 受 履行輔助人之 過失 視同自己過失 (§224)；§227-2 情事變更 |
| 給付有 瑕疵 (瑕疵給付) 或 加害 (加害給付) | 給付物 / 給付行為 不符 債之本旨；或 給付過程 對債權人 其他法益 造成損害 | 完全依約給付，且給付過程未生額外損害；瑕疵 已於 受領時 經債權人 默示同意 | §354 物之瑕疵擔保 (買賣)；§492 工作物瑕疵 (承攬)；瑕疵 vs 給付不能 (§226) 之區辨 |
| 損害 之發生 | 因 不完全給付 而 實際發生 之 財產 / 非財產損害 | 瑕疵存在但 債權人 無實際損害 (e.g. 已修復且未支出費用) | §216 所受損害 + 所失利益；§227 II 加害給付 之 履行利益以外 損害 (e.g. 瑕疵商品 爆炸 燒毀 其他財物) |
| 因果關係 | 不完全給付 與 損害 間具 相當因果關係 | 損害肇因於 債權人 自己 操作不當；第三人 獨立行為 介入 | §217 過失相抵 |

**§227 路徑速記**：

- §227 I 瑕疵給付：給付不符債本旨 → 履行利益賠償 (修補 / 減價 / 解除契約)
- §227 II 加害給付：給付過程另造成 履行利益以外 損害 → 與 §184 競合 (學說：通說 競合，債權人擇一)
- 區辨 §226 給付不能：§227 是「給付了，但有瑕疵或加害」；§226 是「根本給付不能」

## 民法 §767 (物上請求權)

所有人 對於 妨害 / 侵奪 / 有妨害之虞 之 第三人，得行使 返還 / 除去
/ 防止 請求權。屬 絕對權 之 物權保護，不以 故意過失 為要件 (與 §184
最大區別)。第 II 項準用於 其他物權 (地上權 / 地役權 / 抵押權 等)。

| 構成要件 element | 1-line 白話 | 典型 反例 | 典型 carve-out |
|---|---|---|---|
| 請求權人 為 所有人 (或準用之 物權人) | 須對 標的物 有 所有權 或 §767 II 準用之 其他物權 | 僅 占有人 (走 §962 占有人物上請求權，不走 §767)；債權人 (e.g. 租賃權人 — 學說：是否得依 §767 II 類推 有爭議) | §962 占有保護請求權 (一年除斥期間)；§821 共有人 各得單獨請求 (但 返還須 為共有人全體) |
| 標的物 為 特定 之 物 | 須為 特定 動產 / 不動產 / 其他物權標的 | 種類之債 之 種類物 (尚未特定)；金錢之債 (一般 不適用物上請求權，走 §179 / §229 等) | 已特定之 種類物 視為 特定物；§828 公同共有物 之 處分 |
| 對方 無權 占有 / 妨害 / 有妨害之虞 | 相對人 對標的物 無 法律上權源 (所有權以外之 占有正權源 — 租賃 / 借貸 / 留置 等) | 相對人 有 有效租約 / 借貸契約 / 留置權；相對人 為 善意取得人 (§801 / §948) | 租約終止後 相對人 仍占有 → 變為無權占有；§949 盜贓遺失物 之 2 年內 回復請求 |
| (返還請求) 標的物 仍存在且可返還 | 標的物 未滅失，物理上 / 法律上 可返還 | 標的物 已滅失 (走 §184 / §179 損害賠償 / 不當得利)；標的物 已 善意取得 (§801 / §948) | §949 盜贓遺失物 2 年內 回復；§952 善意占有人 之 使用收益 不負返還 |
| (除去 / 防止請求) 妨害 / 妨害之虞 客觀存在 | 對 所有權 行使 之 客觀妨害 (e.g. 越界建築) 或 即將發生 之 妨害 | 主觀感受不適 但 無客觀妨害；忍受義務範圍內 (§793 不可量物 之 通常忍受) | §148 權利濫用 / 誠信原則；§796 越界建築 之 移去義務減免 (學說：經濟成本懸殊 採 利益衡量) |

**§767 路徑速記**：

- 不問 故意過失 — 與 §184 最大區別；只要 客觀 無權占有/妨害 即成立
- 與 §184 / §179 可 並存 — 物權保護 + 損害賠償 + 不當得利 三軌並行 (學說：通說採 請求權競合)
- §767 II 準用範圍 — 其他物權 (地上權 / 地役權 / 抵押權 等)；債權 是否準用 學說對立

## 民法 §179 (不當得利)

無 法律上原因 而 受利益 致 他人受損害 者，應返還其利益。屬 衡平
法理 之 一般 矯正性請求權，與 §184 / §227 / §767 可 並存。學說 區分
為 給付型 (因 給付行為 而生) 與 非給付型 (因 侵害 / 費用 / 求償 等
其他事由)，舉證責任 略有不同 (學說：給付型 vs 非給付型 二元說 通說)。

| 構成要件 element | 1-line 白話 | 典型 反例 | 典型 carve-out |
|---|---|---|---|
| 一方 受 利益 | 受領人 之 財產 因 此事由 而 增加 (積極得利) 或 應減少而未減少 (消極得利) | 雖有交付 但 同時 履行 等值對待給付 (財產未淨增加)；單純 占有 而 未取得 任何法律上 利益 | §181 但書 利益 已不存在 之 善意 受領人 免返還；§182 II 惡意受領人 加計 利息 |
| 他方 受 損害 | 主張權利者 之 財產因此 減少 (積極損害) 或 應增加而未增加 (消極損害) | 雖有給付 但 主張權利者 自始 無權利可損 (e.g. 並非自己之物)；損害 與 利益 無 因果直接關係 | 損害 與 利益 不必 等值；§181 返還範圍 以 利益現存範圍 為限 |
| 利益 與 損害 之間 有 因果關係 | 利益增加 與 損害發生 出於 同一事由 (學說：直接因果關係說 通說 vs 非統一說) | 第三人 介入 行為 造成 損害，與 受領人 利益 出於不同事由；間接因果鏈過長 | 給付型 不當得利：給付行為 即為 因果連結，無須再證明 |
| 無 法律上原因 | 受領利益 欠缺 有效 之 法律上原因 (契約 / 法律規定 / 法院判決 等) | 有有效契約 (e.g. 買賣 / 贈與 / 借貸)；受領 出於 法定債之關係 (e.g. 親屬扶養 §1114)；§180 各款 不得請求返還事由 | §180 (1) 履行 道德上義務；(2) 期前清償；(3) 明知無債務之清償；(4) 不法原因之給付 (e.g. 行賄、賭債) |
| (返還範圍) 受領人 善意 / 惡意 之 區分 | 善意受領人 以 現存利益 為限；惡意受領人 應 返還全部 + 加計 利息 + 損害賠償 | 善意受領人 利益 已不存在 (§181 但書) → 免返還；惡意 之 證明 由 主張權利者 負擔 | §182 II 惡意受領人 之 加重責任；§183 第三人 受讓 之 返還責任 (學說：§183 解釋有 限縮說 vs 通說) |

**§179 路徑速記**：

- 給付型不當得利：因 給付行為 (e.g. 誤匯款 / 契約無效 後給付) — 因果關係 內含於 給付行為
- 非給付型不當得利：因 侵害 / 費用 / 求償 等事由 — 須額外證明 因果關係 (學說：非統一說 採類型化處理)
- §180 各款 為 不得請求返還 之 列舉事由 — 反例命中 即 不該當 (counterfactual.md ratchet 至 不該當)
- 與 §184 / §227 / §767 競合 — 通說採 請求權競合，主張權利者 擇一 行使

## 跨條文交互 速記 (for `counterfactual.md` reverse checks)

| 競合情形 | 通說處理 | 反事實 觸發 |
|---|---|---|
| §184 + §227 II (加害給付) | 請求權競合，擇一行使 | 若 §184「權利」涵攝 ⚠️，可改走 §227 II 確認 (學說：通說 競合) |
| §184 + §179 (不當得利 vs 侵權行為) | 競合，擇一 | §184 若 因 故意過失 涵攝 ⚠️，可改走 §179 (不需 故意過失) |
| §767 + §184 + §179 (物權三軌) | 並存 並行 | §767 不問 故意過失 — 若 §184 在 故意過失 卡關，§767 仍可成立 |
| §227 vs §226 (給付不能) | 區辨：給付了但有瑕疵 (§227) vs 根本不能給付 (§226) | 反例命中 §226 路徑 → §227 不該當 |
| §184 II 違反 保護他人 法律 路徑 | 推定過失 + 不限「權利」 | 個資外洩 / 食安 / 勞動 案件 優先考慮 此路徑 而非 §184 I 前段 |

## 學說對立 inline 標記彙整 (for `subsumption.md` ⚠️ rule)

下列 元素 預期 涵攝結論 為 ⚠️ (除非 事實 squarely 一邊倒)：

- **§184 I 前段「他人之權利」**：個資 / 純粹經濟損失 / 第三人侵害債權 — 學說對立
- **§184 I 後段 vs II 項 vs 第 I 項前段** 之 hypothesis 選擇 — 學說採三 hypotheses 說
- **§184 II 保護他人法律 之範圍**：通說 vs 限縮說 — 行政管制法令 是否 一概包含
- **§227 II 加害給付 與 §184 競合**：通說 採 競合 (學說對立 已弱)
- **§767 II 準用範圍**：是否 及於 債權 (租賃權 等) — 學說對立
- **§179 給付型 vs 非給付型 因果關係**：直接因果說 vs 非統一類型化說
- **§796 越界建築 移去義務 減免**：採 利益衡量 學說，事實敏感

`subsumption.md` 看到 上列 元素 + 事實 不 squarely 一邊倒 時，應
mark ⚠️ + 信心 中 + 註明 「學說對立」，由 `counterfactual.md` 進行
反事實 reverse check。
