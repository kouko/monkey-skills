# 構成要件 — 勞動

涵蓋 **勞基法 §11 (合法解雇事由) / §14 (勞工終止事由) / §16 (預告期間) + 性平法 §13 (性騷擾雇主防治義務)** 四個高頻條文之 構成要件 element list，供 `protocols/subsumption.md` 涵攝 + `protocols/counterfactual.md` 反例 / carve-out 查找。

每條列出構成要件 → 1-line 白話 → 典型 反例 → 典型 carve-out / 抗辯。**條文 text 不 inline**，請 reference [`legal-toolkit/scripts/canonical/legal-sources.json`](../../../scripts/canonical/legal-sources.json) (勞基法 pcode = `N0030001`；性平法 = 性別平等工作法，pcode = `N0030014`)。

> Path A 約定：使用 **勞基法 §11 / §14 / §16** 與 **性平法 §13** 之台灣現行用語，避免 Anglo employment law 詞彙 (e.g. 「wrongful termination」 → 用 `不當解雇` / `違法解雇`；「at-will employment」於台灣不適用)。

## 勞基法 §11 (合法解雇事由 — 雇主終止)

§11 為雇主**單方終止勞動契約**之合法事由 (預告解雇)。共 **5 款** (民國 109 年修正後現行版本)：(1) 歇業或轉讓 (2) 虧損或業務緊縮 (3) 不可抗力暫停工作 1 個月以上 (4) 業務性質變更，有減少勞工之必要，又無適當工作可供安置 (5) 勞工對於所擔任之工作確不能勝任。

涵攝時，每款獨立檢驗 — 各款互不重疊 (學說：吳姿慧 / 邱駿彥 — 各款屬「列舉式」非「例示式」，雇主不得以未列事由解雇)。雇主援用 §11 須**附預告期間** (見 §16) 與 **資遣費** (勞退條例 §12)。

| 構成要件 element | 1-line 白話 | 典型 反例 | 典型 carve-out / 抗辯 |
|---|---|---|---|
| §11①: 歇業或轉讓 | 公司整體停業 / 主體變更而新雇主不留用 | 部分業務外包但公司未歇業 → 不該當 (應走 §11④) | 企業併購法 §16 — 留用協商完成則無 §11① 適用 |
| §11②: 虧損或業務緊縮 | 連續財務虧損或營業額顯著萎縮 | 單季虧損但年度仍盈餘 → 不該當 (學說：須「相當期間」連續虧損；最高法院 100 台上 1057 號) | 雇主已採其他成本控制措施 (減班 / 凍薪) 仍不足 → 強化該當；單純獲利下降非「虧損」(學說：李惠宗 vs 林更盛) |
| §11③: 不可抗力暫停工作 1 個月以上 | 天災 / 戰爭 / 疫情導致工作停止滿 1 個月 | 暫停 3 週後恢復 → 不該當 (時間門檻硬性) | COVID-19 期間「停業命令」屬之 (勞動部 109.04.10 函)；但若僅「自主停業」非主管機關命令，學說對立 |
| §11④: 業務性質變更，有減少勞工之必要，又無適當工作可供安置 | 業務轉型導致原職位消失，且公司內無其他職可調 | 雇主未盡安置義務 (e.g. 直接資遣未先協商調職) → 不該當 (最高法院 96 台上 2630 號「最後手段性原則」) | 安置義務範圍：(a) 同公司其他部門 (b) 關係企業 — 學說對立 (邱駿彥：含關係企業 vs 林更盛：限同公司)。⚠️ Phase 4.5 確認最新判決 |
| §11⑤: 勞工對於所擔任之工作確不能勝任 | 客觀能力不足 OR 主觀消極怠工 | 績效考核 1 次未達標即解雇 → 不該當 (須「持續性」+ 已給改善機會；最高法院 95 台上 916 號) | PIP (Performance Improvement Plan) 已執行且勞工仍未達標 → 強化該當；雇主未事先告知考核標準 → 削弱該當 |

**§11 共通 carve-out**：(a) **解雇最後手段性原則** — 雇主須證明已無其他較輕微手段可採 (學說 通說 + 最高法院定式)；(b) **§13 工資保護期** — 勞工於分娩前後停止工作期間 (產假 / 陪產假) + 職業災害醫療期間，雇主原則不得依 §11 解雇 (例外：天災事變且報主管機關核定)。

## 勞基法 §14 (勞工終止事由 — 勞工不經預告終止)

§14 為勞工**單方不經預告**終止勞動契約之事由 (常稱「被迫離職」/「擬制解雇」)。共 **6 款**：(1) 雇主於訂立勞動契約時為虛偽意思表示 (2) 雇主、雇主家屬、雇主代理人對於勞工，實施暴行或有重大侮辱之行為 (3) 契約所訂之工作對勞工健康有危害之虞，經通知雇主改善而無效果 (4) 雇主、雇主代理人或其他勞工患有法定傳染病，對共同工作之勞工有傳染之虞，且重大危害其健康 (5) 雇主不依勞動契約給付工作報酬，或對於按件計酬之勞工不供給充分之工作 (6) 雇主違反勞動契約或勞工法令，致有損害勞工權益之虞。

勞工依 §14 終止時 **得請領資遣費** (§14 末段準用 §17)，效果上接近 §11 雇主解雇但反向。涵攝時注意 **§14-2 除斥期間 30 日** (自知悉之日起算，①③⑤⑥不適用)。

| 構成要件 element | 1-line 白話 | 典型 反例 | 典型 carve-out / 抗辯 |
|---|---|---|---|
| §14①: 雇主訂約時虛偽意思表示 | 招聘時欺騙 (薪資 / 職務 / 福利) | 招聘時為「期望值」非具體承諾 → 不該當 | 須「致勞工誤信而有受損害之虞」(條文末句) |
| §14②: 暴行 / 重大侮辱 | 主管動手或公開人身攻擊 | 一般工作糾紛 / 嚴詞批評 → 不該當 (須「重大」侮辱) | 「重大」之認定：學說對立 (邱駿彥：客觀標準 vs 林更盛：考量勞工主觀感受)；§14-2 30 日除斥期間 |
| §14③: 工作環境危害健康，通知未改善 | 職安法等具體危害已通知雇主未改善 | 勞工未先通知即逕行終止 → 程序不該當 | 通知形式：書面 / 口頭皆可 (學說 通說)；改善期間「合理期間」由法院個案認定 |
| §14④: 共同工作者法定傳染病有傳染之虞 | 同事染傳染病病重大危及自身 | 一般感冒 / 季節性流感 → 不該當 (須「法定傳染病」) | 雇主已採適當防護措施 → 削弱該當 |
| §14⑤: 雇主不依約給付報酬 | 欠薪 / 短付 / 拖延 | 偶發 1 日延遲且立即補發 → 通常不該當 (學說：須「相當期間」或「重複」)；但 ⚠️ 學說對立 (邱駿彥：1 日即可 vs 林更盛：須持續) | 雇主因不可抗力 (天災) 暫無法給付且即時補發 → 抗辯 |
| §14⑥: 雇主違反勞動契約或勞工法令致勞工權益受損之虞 | 違反工時 / 休假 / 安全衛生規定 | 輕微違規且立即改正 → 不該當 (須「致有損害之虞」) | 「之虞」標準：學說 通說採客觀危險判斷；§14-2 30 日除斥期間 |

**§14 共通 carve-out**：勞工依 §14 終止後請求資遣費，但若**事由不該當**，原雇主可主張為勞工片面**離職** (民法 §488)，無資遣費。Phase 4.5 GC 應確認 §14-2 除斥期間在 ②④ 之適用 (法條僅排除 ①③⑤⑥)。

**§11 vs §14 對照** (subsumption 階段防止當事人混淆)：

| 維度 | §11 (雇主終止) | §14 (勞工終止) |
|---|---|---|
| 終止主體 | 雇主 | 勞工 |
| 預告義務 | 雇主應預告 (§16) | 勞工不經預告 |
| 資遣費 | 由雇主給付 | 雇主仍應給付 (§14 末段準用 §17) |
| 除斥期間 | 無明文，但「最後手段性」隱含合理期間 | 30 日 (§14-2，限 ②④) |
| 救濟若不該當 | 解雇無效 → 勞工得請求恢復僱傭 (民法 §487) | 視為勞工自行離職 → 無資遣費 |

## 勞基法 §16 (預告期間)

§16 規範**雇主依 §11 或 §13 但書解雇**時應給予之**預告期間** + **謀職假**。**注意：§12 (勞工重大違規之懲戒解雇) 雇主不須預告**。

預告期間依勞工**繼續工作年資**分 3 級：

- 繼續工作 3 個月以上 1 年未滿 → **10 日前** 預告
- 繼續工作 1 年以上 3 年未滿 → **20 日前** 預告
- 繼續工作 3 年以上 → **30 日前** 預告

此外：(a) 預告期間內，勞工得於工作時間請假外出謀求新工作，每星期不得超過 2 日工作時間 (謀職假，工資照給)；(b) 雇主未依規定預告而終止契約者，**應給付預告期間之工資** (預告工資代償 — 條文末段)。

| 構成要件 element | 1-line 白話 | 典型 反例 | 典型 carve-out / 抗辯 |
|---|---|---|---|
| 雇主依 §11 或 §13 但書終止 | §16 適用前提是雇主合法解雇且**非 §12 重大違規** | 雇主依 §12①~⑥ 解雇 → §16 不適用 (無預告義務) | §12 解雇須符合 30 日除斥期間 (§12-2)；雇主以 §12 名義但實為 §11 → 法院可能改適用 §11 + 預告義務 |
| 預告期間符合年資門檻 | 3 個月-1 年→10 日 / 1-3 年→20 日 / 3 年+→30 日 | 工作 5 年僅預告 10 日 → 不該當 (應 30 日) | 雇主以**預告工資代償** (條文末段) — 不預告但給付對應日數工資 → 適法 |
| 謀職假提供 | 預告期內每週最多 2 日有薪謀職假 | 雇主拒絕謀職假請求 → 不該當 | 勞工自願不請 → 雇主無代償義務 (學說 通說) |
| 預告工資代償 (反例對應) | 未預告者，應給對應日數工資 | 雇主未預告且未代償 → 違法解雇 (勞工得依 §14⑥ 反向終止 + 求償) | 預告工資按勞工**平均工資** (§2⑤) 計算；爭議時 ⚠️ 確認最新判決對「平均工資」基期之解釋 |

**§16 vs §12 比較表** (subsumption 階段最常見混淆點)：

| 條文 | 解雇性質 | 預告義務 | 資遣費 |
|---|---|---|---|
| §11 (5 款合法事由) | 經濟性 / 業務性 | **YES** (§16 預告 or 預告工資代償) | YES (勞退條例 §12) |
| §12 (6 款重大違規) | 懲戒性 / 勞工有責 | **NO** | NO |
| §13 但書 (天災事變報准) | 經濟性 (例外) | YES (§16) | YES |

## 性平法 §13 (雇主性騷擾防治義務)

性平法 = **性別平等工作法** (民國 112 年修正，pcode `N0030014`)。§13 規範雇主對於工作場所性騷擾事件之**防治義務**，違反時雇主應負**連帶損害賠償責任** (§27)。

§13 構成要件可拆為 3 階段：(a) 工作場所性騷擾事件**發生** (b) 雇主**未採行有效之糾正及補救措施** (c) 受害人**受有損害**。注意 112 年修法後新增 §13 之 1 (受害人未通報且雇主不知之 carve-out) + §13 之 2 (權勢性騷擾加重)。

| 構成要件 element | 1-line 白話 | 典型 反例 | 典型 carve-out / 抗辯 |
|---|---|---|---|
| (a) 工作場所性騷擾事件發生 | 性平法 §12 定義之敵意環境 / 交換利益型騷擾 | 一般職場糾紛 / 言語衝突無性意涵 → 不該當 | 「工作場所」採功能性認定 — 含通勤 / 出差 / 應酬 (學說：劉志鵬 通說) |
| (b) 雇主未採行有效之糾正及補救措施 | 接獲申訴後未即時調查 / 未隔離 / 未懲處 | 雇主 24 hr 內啟動調查 + 隔離加害人 → 不該當 (即時處置抗辯) | 「有效」之認定：學說 通說採客觀標準 + 個案認定；雇主僅「形式調查」(走過場) 不算有效 |
| (c) 受害人受有損害 | 精神 / 身體 / 財產損害 (含離職損失) | 受害人未受實質損害 → §27 連帶賠償不該當 (但 §13 防治義務違反仍可能行政罰) | 慰撫金額通常 NT$50,000 - NT$500,000 區間 (Phase 4.5 確認近年判決趨勢) |

**§13 共通 carve-out**：

- **§13 之 1 (112 年新增)**: 受害人**未通報** (e.g. 私下隱忍) 且雇主**客觀上無從知悉** → 雇主免責。但雇主若**未建立通報管道** (公告 / 訓練 / 申訴信箱) 即無法主張此抗辯 — 法院常認定「應知而不知」。Phase 4.5 GC 應確認最新函釋對「客觀無從知悉」之認定基準 (尤其 LINE / Slack 內部通報是否充足)。
- **§13 之 2 (112 年新增 — 權勢性騷擾)**: 加害人為**雇主本人或代表雇主行使管理權之人**時，**雇主免責抗辯不適用**，責任加重。權勢性騷擾另適用更長之申訴時效 (§32-1：2 年內 / 知悉 1 年內)。
- **規模門檻**: §13 適用對象為**僱用 30 人以上**之雇主應訂定**性騷擾防治措施、申訴及懲戒辦法**並公開揭示 (§13 I)；30 人以下雇主僅有**糾正及補救義務** (§13 II)，但**不免除 §27 連帶賠償** (學說：劉志鵬 vs 林更盛 — 主流說採全面適用 §27)。
- **行政罰平行**: 違反 §13 防治義務除民事 §27 連帶賠償外，另有 §38-1 行政罰 (新台幣 2 萬-100 萬，112 年修法後加重)。subsumption 階段宜同時提示。

## 跨條文交互作用 (subsumption.md 應警示)

| 情境 | 適用條文 | 注意點 |
|---|---|---|
| 雇主以「不能勝任」名義解雇實為性騷擾報復 | §11⑤ 表面 + 性平法 §36 (報復禁止) | 法院常以 §36 反推 §11⑤ 不該當；⚠️ 勞工得依 §14⑥ 反向終止 |
| 雇主依 §12 重大違規解雇但未走 §70 工作規則程序 | §12 vs 工作規則程序瑕疵 | 程序瑕疵不必然致解雇無效，但削弱該當；學說對立 (邱駿彥：實質 vs 林更盛：程序) |
| 性騷擾事件發生但勞工先離職再申訴 | 性平法 §13 + §27 | §13 之 1 carve-out 可能適用，但若雇主早知未處理仍應賠償 |
| 雇主「業務調動」實為變相解雇 | §11④ vs 調動五原則 (§10-1) | 違反五原則 → 調動無效，勞工得依 §14⑥ 反向終止 |
