# Phase 4.5 上市櫃 Compliance 訪談前置作業

> 這個目錄不是 research note 本身，而是「動研究前先把約人 + 問題 + 自己能做的功課準備好」的工作包。

## 為什麼有這個目錄

Phase 4.5（`ROADMAP.md` 中段）是 Phase 5 兩個 skill（`legal-corporate-governance` + `legal-dd-quickscan`）的硬阻塞前置研究。10-15 天工作量裡，**訪談 3-5 位上市櫃 in-house GC 是 lead-time critical path** — 約人 + 排期至少 2-4 週。先把訪談材料準備好，研究才不會卡在「等人答應」。

完整 outline 在 obsidian vault：`2026-05-09 上市櫃 in-house Compliance 工作流深度研究——大綱與議題清單.md`（506 行）。本目錄是該 outline 蒸餾出的可執行子集。
