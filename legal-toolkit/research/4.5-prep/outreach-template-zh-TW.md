# 冷接觸 outreach template（zh-TW）

> 三套 template — 依場景挑用。共同原則：**positioning 定位清楚（開源工具 / 非販售 / 不打廣告）、時長明確（30 min）、利益對等（事後分享 anonymized findings + 工具早期試用）、匿名化保證**。

---

## Template A — LinkedIn / Email 冷接觸（最常用）

```
標題：30 分鐘訪談請益 — 上市櫃 in-house 法務工作流研究

[姓名] 法務長 / GC 您好，

冒昧寫信。我是 [kouko / 個人簡短背景一句]，目前在做一個**開源法務工具研究**，預計
寫成一份 1500-2000 行的「上市櫃 in-house Compliance 工作流深度研究」筆記，做為
個人專案 legal-toolkit（GitHub 開源，MIT 授權）下一階段 sub-skill 設計的 prerequisite。

研究的盲點是：既有公開文獻能告訴我「該做什麼」（公司法 / 證交所規範 / 函令），
但**「實際怎麼跑、痛點在哪、工時花在哪、tooling 缺什麼」這層只有現職 in-house 才知道**。

想跟您請教 30 分鐘（線上或當面皆可），主要聚焦 5 個面向：
1. 一週工時分布（合約 vs Compliance vs 揭露）
2. 重大訊息「即時 / 重大 / 一般」內部判斷的 SOP 還是 case-by-case
3. 法務 vs 法令遵循 / 內部稽核的分工邊界
4. Template / SOP 更新節奏
5. 您最想要的「工具空缺」是什麼

**保證**：
- 開源工具研究，**非販售、非顧問業務、不收費也不會推銷任何產品**
- 訪談內容**完全匿名化**，research note 不會出現您姓名 / 公司；如必要引用會用
  「某 mid-cap 製造業 GC 表示…」形式
- 錄音僅供我逐字稿轉錄使用，研究完成後會刪除原音檔
- 事後會把 anonymized research note 草稿發回給您過目，您可以否決任何引述

**對您可能的價值**：
- 拿到一份 1500-2000 行針對上市櫃 in-house 法務的工作流研究（即將開源）
- 開發中工具的早期試用權（如有興趣）— 也歡迎您提工具 wishlist

不方便也完全沒關係，希望這封信不冒昧。

時間：[兩個候選時段 / 或請您方便時段]

[kouko 簽名 + GitHub repo URL: https://github.com/kouko/monkey-skills]
```

## 候選 GC 來源（建議優先序）

| 來源 | 強度 | 起手式 |
|---|---|---|
| 既有人脈介紹 | ★★★★★ | Template B；接受率最高 |
| LinkedIn — 搜「法務長」「General Counsel」+「上市」 | ★★★★ | Template A；先確認 mid-cap 規模 |
| 律師朋友圈介紹（BigLaw 通常認識所有 in-house GC） | ★★★★ | Template B（先請律師幫忙轉介） |
| 台北律師公會 / TLAA / TBA 在職進修活動會後 | ★★★ | 線下名片交換，事後 follow-up Template A |
| LinkedIn 上市櫃法務群組（如有） | ★★ | Template C 發在群組 |
| 法律科技社群（律果 / LegalSign.ai 社群 / FB 法律科技社團） | ★★ | Template C |

## 數字 target

- **接觸名單**：20 人
- **回覆**：8-10 人（cold outreach 接受率 30-50%）
- **完成訪談**：3-5 位 in-house GC + 1-2 位 BigLaw + 1-2 位法遵 ≈ **5-9 位**
- **訪談排程**：所有訪談排在 **2-4 週內**完成（不要拖；同步研究效率高）

## 回覆無聲時的處理

- 7 天無回 → 一次禮貌 follow-up（「上次的訊息可能被埋了，再次冒昧」）
- 14 天無回 → 標記為 "no response"，不再追
- 明確拒絕 → 道謝，問「方便推薦一位適合的同行嗎？」（雪球效應）

## 訪談後 follow-up SOP

1. **24 小時內**寄感謝信 + 重申匿名化承諾
2. **1 週內** Whisper / metr 轉逐字稿，寫訪談紀要（標記直接引述 vs 概念整理）
3. **research note 草稿完成後**寄 anonymized 引述段給受訪者過目（給 1 週時間）
4. **research note 公開後**寄一份給受訪者（PDF 或連結）
