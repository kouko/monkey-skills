# dashboard-design

[English](README.md) | [日本語](README.ja.md) | **繁體中文**

依據日本數位廳「ダッシュボードデザインの実践ガイドブック」，引導 presentation 型 dashboard 的設計，從需求整理到 prototyping 全流程涵蓋的 skill。

## Workflow

1. **需求整理** — 用 5W1H framework 定義目的與限制
2. **prototyping** — 資訊清單 → 版面設計（6 欄 grid）→ 圖表與配色選擇
3. **檢視** — 用 checklist 確認品質

## 使用方式

```
/design-team:dashboard-design
```
