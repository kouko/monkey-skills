### 佐藤可士和 — UNIQLO LifeWear (JP | Q4 toward-Q1, named AD recast)

## Voice direction
**What this register achieves**: 視覚先行・コピー最小の構造で、服を部品化し生活の OS として再定義する。「整理術」「本質抽出」の手続きがそのまま文体に現れ、短句を句点で切り落としてコピーがロゴのように機能する状態を作る。無印 原研哉 の和的・生活情緒 minimalism と対照的に、グローバル・機能・記号としての minimalism を標榜する。

## Native critical read
- 「整理術 / 整理する」(佐藤自述、著書『佐藤可士和の超整理術』— 情報を引き算し、本質を残す設計手続き)
- 「本質抽出 / ブランドの本質」(佐藤自述、各種インタビュー — 分析と可視化の核語)
- 「簡潔化 / 引き算の編集」(AXIS / BRUTUS 批評語 — 文体評価軸)
- 「記号化 / ロゴ化」(pen / ブレーン — コピーを視覚要素と等価に扱う操作)
- 「部品 / OS / インフラ」(UNIQLO 内部語彙を佐藤が外部化 — 服を生活構造の構成要素として扱う語)

## Prose mechanics
- **体言止め＋句点の「記号化」**：短句を句点で切り落とす。「部品としての服。」「普通であること。」のように、名詞句を完結文として据える。コピーがロゴのように機能する状態。
- **「〜ではなく、〜」の転倒レトリック**：「服に個性があるのではなく、人に個性がある」型。通念を否定 → 本質を提示の二段構造。断定は強いが装飾語ゼロ。
- **連鎖する動詞句の「三段加速」**：「服を変え、常識を変え、世界を変えていく。」——同一動詞の反復で対象のスケールだけを段階的に拡張。形容詞による増幅ではなく、同一構造の反復によるスケール移行。
- **カタカナ造語の「概念固有名化」**：「LifeWear」は大文字始まりの固有名詞として運用される。和文中でも翻訳されず、概念そのものの記号として据え置かれる。原研哉の漢語的抽象（「素」「白」）と対照的に、英語造語を記号ブロック化。
- **「簡潔化」の逆接副詞運用**：「シンプルだが、発想がある。」「普通であること。それが一番難しい。」——平易さと深度を逆接で接続。「シンプル＝安易」の解釈を先回りで潰し、簡潔化こそ設計の到達点と位置付ける。
- **主語省略による「普遍話法」**：「あらゆる人の、あらゆる日常のために。」——主語も述語動詞もない。for 構文の前置詞句だけで文を成立させ、発話主体を消す。広告主＝佐藤＝読者の境界が溶け、ブランド声明が非人称の公共文として響く。

## Don't / Over-mimic
- **Failure mode**: LLM が体言止め＋句点を装飾として濫用、原研哉風「和的 minimalism」と混同し「生活情緒」に流れる。または「シンプル」を形容詞で濫用する。佐藤は「シンプル」を結論語としてのみ置き、プロセス語として使わない。
- **Mitigation**: 「体言止めは整理の帰結としてのみ; 生活情緒語禁止; シンプルは結論語だけ」
