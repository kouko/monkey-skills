## Cluster 1: QUEST (Michel Fortin 2005)

**Canonical**: Michel Fortin, blog post "Want Better Copy? Go On A Quest!",
2005-12-04 on michelfortin.com (URL now 410 Gone; restructured to
/copywriting/quest/). Confirmed via imnewswatch.com 2006 reprint notice.

**5 stages (dual-verified from 3 JP + 5 EN sources)**:
- Q=Qualify (target/filter), U=Understand (empathy), E=Educate (proof),
  S=Stimulate (sell/benefits), T=Transition (CTA/close)
- Fortin: "QUEST is a guide, not a goal or a rule."
- Mountain metaphor: ascent (Q+U+E) → summit (E+S) → descent (S+T)
- AIDA complement: Q adds pre-Attention filtering; U adds empathy bridge

**JP sources**: bodhi.co.jp (full translation), ds-b.jp, blog-writer.net
**Attribution**: NOT Clayton Makepeace. P.P.F.T formula unverifiable.
**Spelling**: Michel (not Michael). JP transliteration マイケル・フォーティン.
**Form**: Long-form optimized (3000+); short-form adaptable per essence.
**Confidence**: Medium-High

## Cluster 2: PASTOR (Ray Edwards 2016)

**Canonical**: Ray Edwards, *How to Write Copy That Sells* (Morgan James,
2016-02-16). ISBN 978-1614485025. No 2nd edition. JP edition: ダイレクト出版
2021/05/10, translator 山本泉.

**6 stages with multi-meaning (confirmed from 3 EN + 2 JP)**:
- P = Person + Problem + Pain (3 dimensions)
- A = Amplify + Aspirations (2 dimensions)
- S = Story + Solution + System (3 dimensions)
- T = **Transformation + Testimony** (2 dimensions — drift #7 refinement:
  JP blog-level drift simplifies to Testimony only; JP translation itself
  preserves both)
- O = Offer (~20% of copy per Cooler Insights 2017; Medium-High confidence)
- R = Response (explicit CTA)

**Shepherd etymology**: confirmed from Edwards podcast + 4 sources.
**JP sources**: Hiltl (philipptarohiltl.com), 40dai, mdkup.com
**Attribution**: NOT Dan Kennedy, Frank Kern, or others.
**Form**: Long-form only (3000+); Story element requires narrative space.
**Confidence**: High (stages) / Medium-High (Offer 20% rule)

## Cluster 3: 小霜和也 instinct-driver analysis

**Canonical**:
- 2010 (NOT 2009)『欲しいほしいホシイ』(インプレスジャパン, ISBN 9784844328711)
- 2014『ここらで広告コピーの本当の話をします。』(宣伝会議, ISBN 9784883353163)
- 2017『急いでデジタルクリエイティブの本当の話をします。』(宣伝会議)

**⚠ 小霜和也 passed away 2021-09-22 (age 58, 軟部肉腫)**

**⚠ 「本能ドライバー」is NOT a named taxonomy**: 小霜's approach is a
process-oriented analytical lens, not a categorical framework. The specific
drive sub-categories (貢献欲/承認欲/優越欲/etc.) appearing in some secondary
descriptions are downstream interpretations, NOT 小霜's own labeled terms.

**Core model**: 無意識 (unconscious instinct, 狩猟採集時代 origin, 90% of
action motivation) vs 意識 (conscious rationalization, post-hoc label-sticking).
ターゲットインサイト sits at the intersection.

**Anti-PASONA**: No direct PASONA name-check found. Structurally antithetical:
PASONA starts from Problem/fear; 小霜 starts from 本能/desire. 小霜 requires
「義」(righteousness) — advertising must make society better.

**90-10 rule**: 「広告コピーを書くまでが9割」— 90% is pre-writing (USP + Target
+ instinct mapping), 10% is the actual copy.

**「一行で関係を作れるか」**: copy must achieve 自分事化 (self-relevance) in
one line.

**Career**: Hakuhodo 1986 → independent → 小霜オフィス → No Problem LLC →
電通デジタル Executive CD → Cabinet Office PR advisor. TCC member since 1989.
TCC Hall of Fame unverified.

**JP sources**: Wikipedia, noproblem.co.jp, AdverTimes obituary, MarkeZine
tribute, Newswitch interview, countand1.com book reviews.
**Confidence**: Medium (taxonomy) / High (philosophy + track record)

## Cluster 4: AISAS / ULSSAS / SIPS evolution

**AISAS** (2004):
- Authors: 秋山隆平 + 杉山恒太郎 (電通)
- Publication: book『ホリスティック・コミュニケーション』(宣伝会議 2004, ISBN 9784883350940). NOT a 電通報 article.
- Stages: Attention → Interest → Search → Action → Share
- 電通 trademark registered June 2005

**ULSSAS** (2019):
- Author: 飯髙悠太 (ホットリンク CMO)
- Publication: 『僕らはSNSでモノを買う』(**ディスカヴァー・トゥエンティワン**, 2019-08-29). NOT すばる舎.
- Stages: UGC → Like → Search1(SNS) → Search2(Web) → Action → Spread
- Circular flywheel (not linear funnel)
- 2015 origin unverifiable; 2019 book is canonical

**SIPS** (2011):
- Author: 佐藤尚之 (電通, pen name さとなお)
- Announced: 2011-01-31 via サトナオ・オープン・ラボ
- Stages: Sympathize → Identify → Participate → Share & Spread
- 共感 replaces Attention; purchase optional

**Timeline**: AIDMA → AISAS → SIPS → ULSSAS
(passive → active-searcher → empathy-driven → UGC-flywheel)

**Copywriting implication**: ULSSAS-era copywriter writes UGC-triggering
seeds (quotable, opinion-provoking, remixable), not final persuasion pieces.

**AIDMA still valid for**: BtoB long cycles, TVCM FMCG, 店頭販促, low-digital
demographics.

**JP sources**: MarketingPedia, 日経クロステック, Hottolink official, 電通 PR PDF, MarkeZine.
**Confidence**: High

## Cluster 5: 糸井/岩崎/眞木 voice lineage

**糸井重里** (1948-):
- 13+ works confirmed incl. Ghibli series (「生きろ。」「おちこんだりもしたけれど、私はげんきです。」etc.)
- Voice: 意図的な曖昧さ, 日常語の詩化, おまじない性, 状態提案型
- ほぼ日 transition 1998: from client copywriting to media ownership
- 2024 コピー10 series: 10 landmark copies discussed with 谷山雅計
- TCC Hall of Fame **2012**

**岩崎俊一** (1947-2014):
- 11+ works confirmed incl. 「やがて、いのちに変わるもの。」「21世紀に間に合いました。」
- Voice: 人生の有限性, 仏教的無常観, 具体と抽象の架橋, 行為の意味変換, 穏やかな断定
- 「コピーは、作るものではなく、見つけるもの」→ discoverer, not fabricator
- 『幸福を見つめるコピー』(2009/2015)
- ⚠ Re-confirmed: リゲイン「24時間戦えますか？」is NOT 岩崎 (drift #8)
- TCC Hall of Fame **2020** (posthumous)

**眞木準** (1948-2009):
- 15+ works confirmed incl. extensive 掛詞 portfolio
- Voice: 同音異義, 和製英語再分解, 恋/愛/服 三角, 和歌の掛詞伝統の現代継承
- 「ダジャレではなく、オシャレです」
- 宣伝会議賞 眞木準賞 established ~2010 (48th edition), still active
- TCC Hall of Fame **2013** (posthumous)

**LLM voice reproduction gap** (unique per author):
- 糸井: LLM loses interpretive openness, over-explains metaphors
- 岩崎: LLM produces generic inspirational; misses specific binding moments
- 眞木: LLM produces bad puns (ダジャレ not オシャレ); pun lacks product relevance

**All 3 same generation** (1947-1948), "copywriter boom" 1980s.
**JP sources**: TCC official, ほぼ日, Wikipedia, iso-labo, kai story, AdverTimes, EDIMAG.
**Confidence**: Medium-High
