# Protocol: Write Short-Form Copy（キャッチコピー Canon short-form writing）

**When to use**: Short-form **キャッチコピー** copy requests. Typical cases — キャッチコピー (advertising headlines), taglines, banner ads, SNS post text, CM titles, magazine / newspaper ad headlines, the single line overlaid on a website's main visual. Word count: **7-15 characters is the default**; 15+ characters should be treated as a lead sentence / subtitle.

**When NOT to use (routing note)**: For **light-action logical-block copy** (50-1,500 characters targeting opt-in / newsletter subscribe / free download / LINE 登録 / LP click-through), do NOT use this protocol's 5 切入点 + 3秒ルール + 7-15 字 constraint. The キャッチコピー form is structurally distinct from light-action logical copy. Light-action workflow uses **PREP 法 / CREMA 法** framework per `../standards/light-action-frameworks.md` — the Light-Action Copy Writing workflow in SKILL.md routes here for structural reference but worker must apply `light-action-frameworks.md` stage definitions (PREP 4-stage or CREMA 5-stage) instead of the キャッチコピー 5 切入点 logic. In practice: if brief is キャッチコピー (7-15 chars), follow this protocol's Phases 1-5. If brief is light-action logical-block copy (50-1,500 chars), reference this protocol's ethics/voice phases only and substitute PREP/CREMA structure for the 5 切入点 approach.

**Output**: A short-form copy artifact within the 7-15 character golden range (for キャッチコピー form) OR a PREP/CREMA-structured light-action copy artifact (for light-action form, per `light-action-frameworks.md`). Select one of the 5 approaches (キャッチコピー) or PREP/CREMA framework (light-action), reference one of the voice masters, and attach "なぜ良いか" 3 items. For キャッチコピー: designed so the reader completes A+I (Attention + Interest) within 3 seconds.

**Grounds on**: `../standards/short-form-catchcopy-canon.md`, `../standards/light-action-frameworks.md`, `../standards/ideation-taniyama-discipline.md`, `../standards/voice-and-tone.md`, `../standards/persuasion-ethics.md`

## Phase 1: 5 Approach Selection

Follow `short-form-catchcopy-canon.md` §5 種切入点決策樹 to select one approach from product + audience + appeal goal.

1. **Organize product / audience / appeal goal**:
   - Confirm the headline seed + "なぜ良いか" 3 items handed off from `copy-ideation-parallel.md` Phase 3 as starting material.
   - Determine whether the product leans solution-oriented or new-category, whether the target is narrow or broad, and whether the reader is ad-savvy.

2. **Selection via decision tree** (`short-form-catchcopy-canon.md` §5 種切入点決策樹):

| Approach | Selection condition | Typical syntax |
|---|---|---|
| **利益 / 願望** | Product is "solution"-leaning + audience responds to positive outcome pursuit | 「〜になりませんか」「〜を手に入れよう」 |
| **恐怖 / 痛点** | Audience responds to negative avoidance appeal (insurance / health / security) | 「このままで大丈夫？」「〜を失う前に」 |
| **顛覆常識** | Product is "new category" + need to wake a dormant audience | 「誰が〇〇は△△と決めた？」 |
| **目標呼喚** | Target is narrow (B2B / niche / specialized service) | 「〜したい、あなたへ」「〇〇世代のための」 |
| **提問互動** | Reader is ad-savvy + curiosity arousal is effective | 「なぜ〇〇は〇〇なのか？」 |

3. **Ethics boundary check** (`persuasion-ethics.md` §Dark Pattern 反模式清單):
   - If selecting 恐怖 approach, pre-check against `persuasion-ethics.md` §景品表示法要點 + §小霜和也「嘘をつかない」原則 to ensure you are not falling into excessive anxiety-based marketing.
   - Pre-check the 小霜 principles in this Phase: "no exaggeration", "no fabricated benefits", "fear is not the base strategy".

4. **Record**: Note the selected approach and selection rationale as meta-description at the top of the draft.

## Phase 2: AIDMA Short-Form Scope Confirmation

Follow `short-form-catchcopy-canon.md` §AIDMA 短文案作用域 to reconfirm the canonical division — **short-form is A+I (Attention + Interest) only**.

1. **A+I sole-charge confirmation**: Verify that the short-form is not trying to cover D (Desire) / M (Memory) / A (Action) on its own.
   - D is handled by body copy
   - M is handled by repeated exposure / jingles
   - A is handled by CTA / LP
   - Cramming all 5 stages into one short-form causes either "over word count (15+ chars)" or "insufficient persuasiveness".

2. **3秒ルール check** (`short-form-catchcopy-canon.md` §3 秒ルール):
   - **1 second**: catch + main visual trigger interest ("What is this?" — stop the reader)
   - **2 seconds**: the core message ("This is relevant to me" — recognition)
   - **3 seconds**: next-action induction ("I want to see more / click")
   - For SNS media (TikTok / Reels / Shorts), shrink to 1-3 seconds.
   - **3秒ルール is a heuristic, not a formula**. Do not force-apply it as a staged design.

3. **AIDMA / AISAS / ULSSAS selection by medium** (`short-form-catchcopy-canon.md` §SNS 時代の演化):
   - B2B / TV CM / in-store / magazines → AIDMA mainstream
   - SNS / e-commerce → AISAS (2004 電通) / ULSSAS (2015 ホットリンク) mainstream
   - Under any framework, the A+I sole-charge scope of short-form copy is invariant.

## Phase 3: Character Count Discipline — 7-15 字 Golden Range

Follow `short-form-catchcopy-canon.md` §7-15 字 黃金範圍 strictly.

1. **7-15 character canonical rationale**:
   - Human short-term memory span (magical number 7 ± 2)
   - Upper limit for fitting on a single smartphone screen line
   - Proximity to the 17-mora haiku sensibility
   - Measured values from 4 masters: 糸井 7 / 岩崎 12-14 / 眞木 10 / Yahoo Topics 13

2. **Boundary handling**:
   - If exceeding 15 characters, treat as "lead sentence / subtitle / body copy".
   - Exceeding is permissible for long-form headlines (e.g., opening of PASONA P stage), but the reason for exceeding must be stated.
   - Sub-7 characters (1-6 chars) are also possible (e.g., "生茶" at 2 chars), but rhythm / meaning density will be scrutinized.

3. **Character counting method**:
   - 1 full-width character = 1 char count (2 half-width characters = 1 char)
   - Punctuation (、。) counts toward character count
   - Symbol / whitespace handling follows the medium's rules (Yahoo! Topics canonical = 13 half-width chars)

4. **When exceeding the count**:
   - Are there words that can be cut? (谷山「磨く」first focus, `ideation-taniyama-discipline.md` §階段 3)
   - Can a single particle character shorten it?
   - Can 2 sentences merge into 1? (punctuation reduction)

## Phase 4: 「なんかいいよね禁止」Review

Apply `ideation-taniyama-discipline.md` §「なんかいいよね禁止」ルール to every candidate.

1. **Mandatory "なぜ良いか" 3 items**: For each adoption candidate X, write in the following format:
   ```
   Candidate: X
   なぜ良いか:
   1. [Concrete reason 1: What is conveyed to whom]
   2. [Concrete reason 2: What is new relative to existing similar copy]
   3. [Concrete reason 3: What resonates in the target's life / context]
   ```

2. **Description vs. solution judgment** (`ideation-taniyama-discipline.md` §描写 vs 解決):
   - Description-type (NG): "この化粧水は保湿力が高い" — mere rephrasing of product feature
   - Solution-type (OK): "明日の朝、鏡を見るのが楽しみになる化粧水" — presents meaning in the reader's life
   - The 3 items must include reasons proving "this is not description, but solution / new meaning."

3. **Candidates for which the 3 items cannot be concretized are not adopted**. Either persist until they are, or discard.

4. **Writer-perspective self-review**: Is it only good because you know your own ideation route? (`ideation-taniyama-discipline.md` §階段 2 tip「自分の最初のお気に入りを疑う」).

## Phase 5: Voice Master Reference

Reference `short-form-catchcopy-canon.md` §四位 voice 巨匠 and `voice-and-tone.md` §JP 情緒共鳴傳統 to **explicitly state whether the candidate belongs to any voice lineage, or intentionally departs from one**.

| Master | Lineage characteristics | Inheritance point |
|---|---|---|
| **糸井重里** | State-proposal / intentional ambiguity / poetic rearrangement of daily vocabulary | State presentation not command, room for interpretation |
| **岩崎俊一** | Life philosophy / Buddhist impermanence / seasonal sensibility | Finitude, life, "discover" philosophy |
| **眞木準** | 掛詞 / phonetics / "not pun but pun-chic" | Homophone play, 和歌-descended sound play |
| **谷山雅計** | Entry design / 31 training exercises | Idea generation as constitution, thorough verbalization |

1. **Reference lineage selection**: If already defined in the voice guide (`voice-and-tone.md` §Brand Voice Guide), follow it. If undefined, consider brand affinity in this Phase.

2. **Avoid directly applying Anglo headline formulas** (`short-form-catchcopy-canon.md` §Anglo vs JP boundary):
   - Do not apply Caples headline types or Ogilvy headline rules directly to JP short-form copy.
   - It produces "translation-toned meaning-explicit" copy that lacks emotional resonance / 掛詞 / 余韻.

3. **Ogilvy headline compression density for reference only** (`voice-and-tone.md` §Ogilvy 長文案 voice 經典原則): The density standard derived from "5x more read the headline" is useful, but do not Anglo-ize the voice itself.

## Rules

- Default to the 7-15 character golden range. Exceeding requires explicit justification (`short-form-catchcopy-canon.md` §7-15 字 黃金範囲).
- Short-form is A+I sole-charge. Do not cram D/M/A into one line (`short-form-catchcopy-canon.md` §AIDMA 短文案作用域).
- 3秒ルール is a heuristic. Do not formula-ize it into stages (`short-form-catchcopy-canon.md` §3 秒ルール canonical note).
- All candidates must have "なぜ良いか" 3 items attached. Candidates where 3 items cannot be written are not adopted.
- Do not stop at description-type copy. Reach solution / new meaning (`ideation-taniyama-discipline.md` §描写 vs 解決).
- Do not attribute リゲイン「24 時間戦えますか？」 to 岩崎俊一. Canonical: 杉山恒太郎 + リゲイン PJ (`short-form-catchcopy-canon.md` §drift #8).
- Do not imitate / adapt TCC yearbook works. "Reference and refine your own voice" is canonical.
- When selecting the 恐怖 approach, verify against `persuasion-ethics.md` §景品表示法要點 + §小霜原則 to avoid excessive anxiety-based marketing.

## Anti-Patterns

- **Calling 15+ characters "キャッチコピー"**: The canonical boundary treats it as a lead sentence / subtitle / body copy (`short-form-catchcopy-canon.md` §Anti-Patterns).
- **Design where the core message does not land within 3 seconds**: Information density is too low to meet the reader's judgment threshold.
- **Force-applying Anglo headline formulas to JP short-form copy**: Destroys emotional resonance / 掛詞 / 余韻.
- **Attributing リゲイン「24 時間戦えますか？」to 岩崎俊一**: drift #8 violation. Canonical: 杉山恒太郎 + リゲイン PJ.
- **Covering all AIDMA stages with a single short-form**: Either word count overflow or insufficient persuasiveness.
- **Formula-izing 3秒ルール into staged design**: It is a heuristic, not a stage formula.
- **Imitating / adapting TCC yearbook works**: Damages voice uniqueness.
- **Treating AISAS / ULSSAS as a complete replacement for AIDMA**: AIDMA remains mainstream in B2B / TV CM / in-store. Select by medium.
- **Adopting candidates via "なんかいいよね"**: Choosing candidates for which the 3-item rationale cannot be written produces non-reproducible coincidence (`ideation-taniyama-discipline.md` §Anti-Patterns).
- **Stopping at description-type copy**: Ending with "○○は△△です". Product feature rephrasing with no new meaning presented.
- **Choosing from the writer's perspective**: Selecting a candidate that only looks good because you know the ideation route.
- **Using superlative claims ("最高", "No.1", "世界初") without evidence in short-form**: 景品表示法 優良誤認 / 有利誤認 frequent violation type. Even under the 15-char constraint, evidence is a prerequisite.
