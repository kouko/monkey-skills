# 小霜和也 Instinct-Driven Copy Analysis

Tier 3 standard: fully self-contained. Codifies 小霜和也's
instinct-driven (本能駆動) approach to copywriting as an analytical
lens for ideation and copy critique. This standard treats 小霜's
method as a **process-oriented analytical tool**, not a categorical
taxonomy — a critical distinction (see Attribution Corrections).

小霜's approach is structurally antithetical to PASONA-family
frameworks: where PASONA starts from Problem (fear/pain), 小霜
starts from 本能 (instinctive desire). Where PASONA prescribes
sequential stages, 小霜 prescribes a pre-writing analytical
process. This standard documents both the method itself and its
positioning relative to the PASONA family.

  secondary sources cite 2009 as the publication year of
  『欲しいほしいホシイ』. The canonical year is **2010**
  (インプレスジャパン).

  小霜's approach is a **process-oriented analytical lens**, not a

  (貢献欲 / 承認欲 / 優越欲 / etc.) appearing in some secondary

## Core model: 無意識 vs 意識

小霜's foundational thesis:

```
無意識 (unconscious instinct)
  ├── Origin: 狩猟採集時代 (hunter-gatherer era)
  ├── Drives ~90% of action motivation
  └── Copy must trigger THIS layer

意識 (conscious rationalization)
  ├── Post-hoc label-sticking on instinctive responses
  ├── Drives ~10% of action motivation
  └── Copy that appeals only to THIS layer → weak response
```

**ターゲットインサイト** sits at the intersection — the moment
where an unconscious instinct surfaces into a recognizable
(but not yet rationalized) feeling. The copywriter's job is to
find this intersection point for a specific target.

The model does not prescribe specific instinct categories.
Secondary sources list sub-categories (貢献欲, 承認欲, 優越欲,
支配欲, etc.), but these are **interpretive derivatives**, not
小霜's canonical terms. Workers should use the 無意識 vs 意識
framework as an analytical process, not a category checklist.

## 90-10 rule

「広告コピーを書くまでが9割」— 90% of the work is pre-writing:

| Phase | Proportion | Content |
|-------|-----------|---------|
| Pre-writing (9割) | ~90% | USP identification + Target definition + instinct mapping (which 無意識 trigger does this product activate for this target?) |
| Writing (1割) | ~10% | The actual copy — which, if the pre-writing is thorough, "writes itself" |

This is structurally different from most frameworks that spend
the majority of their guidance on the writing stages. For 小霜,
if you're struggling to write the copy, the problem is in the
pre-writing analysis, not the writing technique.

### Pre-writing process

1. **USP extraction**: What does this product do that no other
   product does? (Functional, not emotional.)
2. **Target definition**: Who is the specific person whose instinct
   this USP could trigger? (Not demographics — psychographic
   instinct profile.)
3. **Instinct mapping**: Which 無意識 drive connects the USP to the
   target? Map the unconscious desire that the product fulfills.
4. **ターゲットインサイト crystallization**: Find the one sentence
   that captures the target's "I've always felt this but never
   said it" moment.
5. **Copy generation**: Write from the ターゲットインサイト. The
   copy should make the reader experience 自分事化 (self-relevance)
   in one line.

## 「一行で関係を作れるか」 — one-line relationship test

Copy must achieve 自分事化 (self-relevance) in one line. If the
reader cannot feel "this is about me" within the first line, the
copy has failed — regardless of how well the subsequent lines are
crafted.

This principle aligns with `short-form-catchcopy-canon.md` §3秒
ルール but adds a deeper criterion: 3秒ルール measures attention
capture; 一行で関係を作れるか measures instinctive self-identification.

## Structural comparison with PASONA family

| Dimension | PASONA family | 小霜 method |
|-----------|---------------|-------------|
| Starting point | Problem (fear/pain) | 本能 (instinctive desire) |
| Structure | Sequential stages (P→A→S→O→N→A) | Analytical process (pre-writing → writing) |
| Primary driver | Conscious problem recognition | Unconscious instinct activation |
| Persuasion mechanism | Escalate problem → present solution → scarcity → CTA | Trigger instinct → 自分事化 → natural action |
| Copy proportion | Entire copy follows stage structure | 90% pre-writing, 10% copy |
| Ethics stance | Affinity replaces Agitation (神田 2016 reflection) | 義 (righteousness) — advertising must make society better |

**No direct PASONA name-check found in 小霜's canonical works.**
The anti-PASONA positioning is structural: 小霜's method starts
from desire (what the target instinctively wants) rather than
fear (what the target is afraid of). This is not a critique of
PASONA but a fundamentally different entry point.

## 義 (righteousness) — ethical foundation

小霜 requires 「義」 in advertising: advertising must make society
better. This is not a compliance rule (like 景品表示法) but a
philosophical stance — the copywriter has a responsibility to use
instinct-triggering power for positive ends.

Practical implications:
- Copy that triggers instincts to manipulate or deceive violates 義
- Copy that triggers instincts to inform genuine choices fulfills 義
- The distinction is not in technique but in intent and outcome

Cross-reference with `persuasion-ethics.md`: 義 operates at a
higher philosophical level than compliance ethics. A copy piece
can pass all ethics checklist items and still violate 義 if its
intent is manipulative.

## Application in copywriting-team workflows

### As an ideation lens

小霜's method serves as an alternative ideation entry point in
`copy-ideation-parallel.md` and the advanced ideation protocol.
Instead of (or in addition to) 曼陀羅 direction-based divergence,
use instinct-mapping as a divergence axis:

1. Map the product's USP
2. Identify 3-5 different 無意識 triggers the USP could activate
3. For each trigger, write candidates that achieve 自分事化
4. Converge through standard KJ + なんかいいよね禁止 process

### As a copy critique lens

When auditing copy (`copy-audit.md`), add an instinct-layer check:
- Does the copy trigger 無意識 or only appeal to 意識?
- Can the target experience 自分事化 in the first line?
- Is the pre-writing (USP + target + instinct mapping) visible
  in the copy's structure, or is the copy structurally arbitrary?

### As a voice reference (with caution)

小霜's own copy style is distinct from the three masters in
`jp-copy-craft-lineage.md`. His voice is more analytical and
direct — reflecting the CD (Creative Director) perspective rather
than the pure copywriter perspective. Use 小霜 as a method
reference, not a voice reference, unless the brief specifically
calls for analytical-direct copy.

## Anti-Patterns

- **Treating 「本能ドライバー」as a fixed taxonomy with named
  categories**. The sub-categories (貢献欲, 承認欲, etc.) are
  downstream interpretations. Use the 無意識 vs 意識 model as a
  process, not a checklist (drift #10).
- **Citing 2009 as the first book's publication year**. Canonical
  year is 2010 (drift #9).
- **Skipping the pre-writing process and jumping to "instinct-
  driven copy"**. The 90-10 rule means the method IS the
  pre-writing. Without USP + Target + instinct mapping, the copy
  is not instinct-driven — it's just copy with instinct vocabulary.
- **Equating 小霜's method with "emotional marketing"**. 小霜's
  instinct layer is deeper than emotion — it targets 無意識
  (unconscious) drives, not 感情 (conscious feelings). Emotional
  marketing operates in the 意識 layer.
- **Using 小霜's method to justify manipulative copy**. 義
  (righteousness) is a non-negotiable ethical foundation. Instinct-
  triggering for deception violates the method's core philosophy.
- **Presenting 小霜 as "anti-PASONA"**. The methods are
  structurally different starting points, not opponents. A
  copywriter can use 小霜's pre-writing process and then structure
  the copy using PASONA stages — the two are complementary at
  different layers.
- **Attributing the TCC Hall of Fame status to 小霜**. Hall of
  Fame membership is unverified. TCC member since 1989 is confirmed.
