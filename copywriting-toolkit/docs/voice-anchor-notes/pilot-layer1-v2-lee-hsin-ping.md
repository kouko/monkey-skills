### 李欣頻 — 誠品副作用 existential-aphorism (zh-TW | Q2 center, craft-gate master)

## Voice direction
**What this register achieves**: 以排比枚舉 + 具名文化座標建構「讀者—作者—品牌」三方已然對等的閱讀場；把購買動作改寫為存在主義式的生活命題，不說服、不教育，只開啟一個「你也在這裡嗎」的識別信號。

**Native critical read**:
- 「誠品副作用」(李欣頻 自述書名 /《誠品副作用》新新聞文化 1998 — 指文案讀完後殘留的 after-effect)
- 「排比枚舉」(craft-lineage 深度檔 pattern 名；即四連 / 五連 / 七連具名名單構成的呼吸節奏)
- 「具名引用 / 文化座標」(craft-lineage — 寺山修司 / 阿莫多瓦 / 徐四金 / 村上春樹 / 彼得梅爾等具名 signifier 承重)
- 「存在主義式格言」(誠品 1990s 文案定位 — 以「所有...都即將在這裡發生」命題句封結)
- 「讀者地位反轉」(李欣頻 自述 — 把讀者當作「已經讀過」的同代人)

## Prose mechanics
- 排比枚舉必須 ≥4 項具名文化座標（人名 / 作品名），通過「抽掉一個名字節奏就壞」承重測試；泛稱如「文學／詩／散文」=voice collapse
- 每篇開頭以「所有 / 每一 / 在...之間」先鋪出命題宇宙，再填內容，避免直進動作句
- 動詞選「繁衍 / 別戀 / 拋開 / 遇見 / 歡迎」承擔情感重量；禁用「買 / 選 / 試 / 享 / 體驗」
- 句尾多用名詞詞組封結（「...的可能」「...的方式」「...的群像」），避免閉合式宣言或 CTA
- 量詞 / 數字做詩意抽換（「九九九九種」「一千零一種」），避免常態「很多 / 多種」
- 全文須與一個可指明的前文本對話（如《書を捨てよ、町へ出よう》寺山修司 1967）— 若無 antecedent work，voice 立即空心仿作
- 排比中人稱保持非個人化（不用「我」「你」），以文化場而非親密關係建立識別
- 禁用說明性語氣（「讓我們告訴你...」「你知道嗎...」）

## Examples
- 「所有創建一座書店的慾望，都即將在這裡發生。」(誠品敦南 ~1995,《誠品副作用》1998 收錄)
- 「送舊迎新．移館別戀」(誠品敦南搬遷 ~1995,《誠品副作用》1998)
- 「九九九九種繁衍生活的創意方式」(誠品敦南新館開幕 ~1995,《誠品副作用》1998)
- 「以慾望別，而非物品別分類。」(誠品品牌定位 1989-1995,《誠品副作用》1998)
- 「拋開阿莫多瓦的高跟鞋到街上去。拋開村上春樹的彈珠遊戲到街上去。拋開徐四金的香水到街上去。拋開彼得梅爾的普羅旺斯到街上去。」(誠品敦南「拋開書本到街上去」系列，致敬寺山修司《書を捨てよ、町へ出よう》1967)
- 「在書與非書之間，我們歡迎各種可能的閱讀者。」(誠品《閱讀者的群像》,《李欣頻的廣告四庫全書》時報 2016)
- 「我在閱讀中，遇見另一個自己。」(誠品書店 canonical,《廣告四庫全書》2016)

## Don't / Over-mimic
- **Failure mode**: LLM 仿作保留排比句型與命題化語氣，但 (a) 具名座標替換成泛稱類別（「小說／詩／散文」），(b) 無 antecedent work 對話，(c)「繁衍／別戀」被當裝飾形容詞而非承重動詞 → 產出 listy rhythm + hollow payload
- **Mitigation**: 強制四個具名文化座標 + 一個 antecedent work；抽掉名字節奏不壞則退回
