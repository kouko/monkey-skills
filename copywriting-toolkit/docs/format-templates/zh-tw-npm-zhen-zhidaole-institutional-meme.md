# 故宮粉絲團「朕知道了」era — institutional SNS format template

**Status**: NOT a voice anchor (v2 inclusion criterion). This is a **format template** documenting institutional SNS register mechanics for reference by Phase 8 form-check and audit skills. Pass 3 MUST NOT load this as a voice anchor.

## Why this is a format template, not a voice anchor

Per `docs/anchor-schema-v2.md` §Inclusion criterion: voice anchor requires individual creator with identifiable sentence-level register. 故宮粉絲團 operates on a **rotating 6-10-person in-house 小編 team** (均齡 26 歲) with a weekly-cycle SOP — the register is the institution's formatted template, not one creator's voice.

⚠ **Z11 Correction preserved (v1.4.0)**:「朕知道了」is **康熙硃批** (NOT 雍正 — v1.3.x 標錯). Source: 2004-09 to 2005-05 馮明珠 curated〈知道了：硃批奏摺展〉(Eng: "Thou Art Understood!"); exhibition brochure cover selects 康熙. 2013-07 紙膠帶 launch re-used this existing IP.

## Era

2013-07-04「朕知道了」紙膠帶 FB 首貼 → sustained humorous register through 2010s-2020s; 2017-19 小編團隊擴編至 6-10 人輪值制.

## Creator

國立故宮博物院 in-house 新媒體小組（均齡 26 歲）+ 文創合作. No isolable individual creator — all decisions are committee + weekly-cycle SOP + rotating-author execution.

## Format mechanics (documentable as template, NOT as voice)

- **讓歷史走進生活**（故宮文創 mission 自述）
- **故事行銷**（小編自述核心策略）
- **深刻的衝突感**（ADM 廣告雜誌 2018 — 博物館莊嚴 × 活潑語氣）
- **故宮小編 / 隆哥 / 波編**（persona 暱稱 sticky label）
- **古物擬人化 / 文物人設**
- **週循環四節奏**（週一興趣 / 週二知識 / 週三導購 / 週四轉話題 SOP）
- **輪值制 6-10 人小編群**

## Measurable register features

- **古文 / 白話比例 ~20-30% / 70-80%**（code-switch 比例精確；「朕」「臣妾」「領旨」直接接「吃太多」「卡住」「高大上」「怪我囉」）
- **Two-speaker dialogue 擬人**（皇 × 后對話），非單 persona 獨白
- Meme = 文物 fragment + 現代 caption **「減法」結構** — 文物本身不改，只加現代 caption
- 每貼對應真實 SKU（commercial ground truth），防止 persona 漂移
- 商品導購反向驅動：節慶/時事 → 從館藏 inventory 找 metaphor（長輩問薪水 → 封條紙膠帶）

## Platform differentiator (vs similar institutional SNS)

故宮獨有 **"institutional IP × commerce × historical corpus" 三層** — 擬人對象是自家文物（非外部 mascot）、古文 code-switch 有文物正當性、每貼文對應真實 SKU. vs:
- 中研院〈研之有物〉純 explainer 無商業
- 北美館 curatorial-neutral 無人設
- 台灣吧 娛樂-only 無商業錨點

## Known failure modes when template is misused as voice

- (a) 古文比例失控（LLM 傾向堆全文言體 >50%）
- (b) two-speaker 退化為單 narrator
- (c) 虛構不存在的館藏文物（真實洩漏風險）

Mitigation for Phase 8 form-check: "古文比例 ≤30%; 不得虛構館藏; 擬人須 two-speaker dialogue".
