### 許舜英 — 意識形態廣告 definitional inversion (zh-TW | Q2 center, craft-gate master)

## Voice direction
**What this register achieves**: 用「定義反轉」把商品文案改寫成文化批判 — 先丟出一個帶權力意味的等式（「X 是一種 Y」），再把整個框架翻轉，逼讀者在面對政治／性別／危險／禁慾等詞彙之後才能進入 pitch。箴言密度承載所有 payload，品牌是「讀懂這個句子」之後的副產品。

**Native critical read** (5):

- 「文化批判 payload underneath definitional inversion」(zh-copy-craft-lineage Drift-Prevention Card)

- 「三日不購物便覺靈魂可憎」(作者自引於《購物日記》開卷 — self-citation as register anchor)

## Prose mechanics (7)
- 使用 `X 是一種 Y` / `X 就是 Y` 定義反轉句型恰好一次作為脊骨；Y 須屬於與 X 不同的權力領域（服裝→政治 / 消費→倫理 / 內衣→辯證）
- 每篇必含至少一個 power-disparity word（政治 / 統治 / 殖民 / 禁慾 / 失敗 / 危險 / 虛妄）；若 draft 缺之，register 即為偽作
- 從哲學 / 政治 / 姿態 / 命題 / 辯證 / 美學 domain 移植詞彙進消費 context；禁用可愛化、diminutives、驚嘆、emoji-adjacent 語氣
- 句末落在名詞或抽象概念（。承擔哲學重量），不落在動詞 CTA（買 / 選 / 試 / 享 disallowed as closers）
- 結構為 assertion → inversion → coda（單一 assertion 即 hollow）；一篇最多一個自答式 rhetorical question
- 整句 declarative、非 fragment（與糸井相反）；散文呼吸節奏靠、和。chain，非 staccato
- 文化引用須具名、非泛稱（梅蘭芳 / 寺山修司 / 祖母衣櫃 / 港式素蠔油）；assumes reader-as-cultural-peer，絕不解釋

## Don't / Over-mimic
- **Failure mode**: LLM 產出空洞的箴言反轉句（「咖啡就是一種生活」「旅行是一種態度」）— 表層文法有複製、文化批判的 payload 卻不見；連續堆疊多個自答的問句；把具名的文化座標泛化成「書／音樂」；只寫 assertion 不做 inversion
- **Mitigation** (≤15 字)：強制每句含 1 個 power-disparity 詞＋1 個具名文化座標＋反轉；缺即退回
