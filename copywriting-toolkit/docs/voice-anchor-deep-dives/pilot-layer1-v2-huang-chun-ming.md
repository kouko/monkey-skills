### 黃春明 — 鄉土文學 humane vernacular (zh-TW | Q3 toward-Q2, author)

## Voice direction
**What this register achieves**: 讓底層小人物用自己的台語腔國語開口說話，敘述者退到一旁只做白描；尊嚴與悲傷不被命名，而是從身體動作、家常對白、與一兩句自嘲裡漏出來。

**Native critical read** (6):
- 「小人物書寫」「悲憫的凝視」(呂正惠《戰後台灣文學經驗》1992)
- 「把小丑的眼淚寫成史詩」(王德威〈黃春明論〉《跨世紀風華》)
- 「鄉土文學」(1977-78 鄉土文學論戰 canonical 術語；陳映真、尉天驄編《鄉土文學討論集》將黃春明列為代表作家)
- 「人道關懷 / 現實主義」(1970s《文季》《夏潮》批評 recurring 用語)
- 「我只是替他們把話說出來」(黃春明 1980 訪談,《八十年代》雜誌)
- 「聽得見台灣話的國語」(侯孝賢 1983 談《兒子的大玩偶》改編)

## Prose mechanics (7)
- **對白領先敘述**：一個場景先讓人物開口三到五句話，敘述句只負責交代動作（「他把斗笠往地上一擱」），不進角色內心
- **台語句法嵌進國語**：用「是按怎」「哪會」「給人笑」「不甘」「來去」替「為什麼 / 怎麼會 / 被笑 / 捨不得 / 過去」；語助詞用「啦、咧、喔、吼」收尾，不用「呢、吧」
- **稱謂貼著階級**：角色互稱「坤樹」「阿盛仔」「白梅」「那個日本仔」，不用「他 / 她」帶過
- **身體動作取代情緒詞**：禁用「難過 / 心酸 / 無奈 / 落寞」；改寫成「蹲在路邊抽菸」「把那張紙摺了又摺」「用袖子抹一下嘴」
- **自嘲句 deflect 悲劇**：每段最沉重處安排一句人物自己的玩笑，苦味從笑裡滲出，不從哭裡流出
- **短景交替**：4-8 行一個畫面，畫面之間不用「後來」「於是」「沒想到」連接，直接換行跳接
- **小人物視角鎖死**：敘述者知道的不多於角色；絕不出現「殊不知幾年後...」「命運就這樣...」全知抒情介入

## Don't / Over-mimic
- **Failure mode**: LLM 容易把「鄉土」做成懷舊濾鏡 — 加入「那個年代的純樸」「熟悉的稻香」「時光彷彿靜止」這類敘述者抒情句，把黃春明的白描庶民變成觀光客式鄉愁，悲憫也滑成廉價感傷
- **Mitigation** (12 words): "No narrator-nostalgia; emotion shows via body action + self-deprecating joke only"
