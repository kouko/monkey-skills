### 張愛玲 Eileen Chang (zh | Q2 toward-Q3)

## Voice direction
**What this register achieves**: 以日常細節的比喻密度，讓讀者在華美表層上突然撞見破敗底色 — aphoristic 觀察的 payload = 美與蝕同時到場，不靠抒情詞完成。

**Native critical read**:
- 「蒼涼」(張自述，《自己的文章》— 自訂美學基調，對立於「悲壯」)
- 「參差的對照 / 蔥綠配桃紅」(張自述，《自己的文章》— 反對斬釘截鐵的衝突，改用不調和的並置)
- 「不徹底的人物」(張自述，《自己的文章》— 拒寫英雄，只寫軟弱時代下的蒼生)
- 「意象繁複在中國現代小說家中首屈一指 / 視覺想像達到濟慈那樣華麗的程度」(夏志清《中國現代小說史》1961)
- 「荒涼 / 頹廢 / 艷異 / 否定的辯證」(王德威 張學 canonical critical cluster)
- 「張腔 / 張愛玲體 / 通感依附於比喻」(學界 + 譚志明《東海中文學報》24 期)

## Prose mechanics
- 日常器物喻抽象命題 — 抽象降落到 household-scale 實物（袍、蚤、蚊子血、飯黏子、蚊煙香盤）；不用宏大隱喻，用具體得可笑的小東西作落點
- 對偶比喻撐起反諷 — 一句內部兩組平行比喻互相 undermine（紅→蚊子血 / 白→飯黏子）
- 感官混接 (通感 + 色彩) — 視覺 × 味覺 × 觸覺混搭（「鮮辣的潮濕的綠色」「白辣辣的雨」）；顏色加味覺形容詞，不加情緒形容詞
- 名詞句收束 — 長鋪排句末落在具體名詞或動作定格（「將蚊煙香盤踢到桌子底下去。」）不收形容詞或抒情短語
- 讓步–逆轉 微句型 — `X 成全了 Y；但⋯誰知道呢？` / `長的是 A，短的是 B。`
- 冷筆寫大事件 — 戰爭、毀城、墜落，用家常動作接住
- 時間尺度 open frame — 「三十年前」「一百年」「一輩子」替代情緒強度詞

## Examples
- 「生命是一襲華美的袍，爬滿了蚤子。」(散文《天才夢》1941)
- 「三十年前的上海，一個有月亮的晚上⋯再好的月色也不免帶點淒涼。」(《金鎖記》1943 開頭)
- 「三十年前的月亮早已沉下去，三十年前的人也死了，然而三十年前的故事還沒完——完不了。」(《金鎖記》1943 結尾)
- 「娶了紅玫瑰，久而久之，紅的變了牆上的一抹蚊子血，白的還是『床前明月光』；娶了白玫瑰，白的便是衣服上沾的一粒飯黏子，紅的卻是心口上一顆硃砂痣。」(《紅玫瑰與白玫瑰》1944)
- 「香港的陷落成全了她。⋯也許就因為要成全她，一個大都市傾覆了。」(《傾城之戀》1943)
- 「長的是磨難，短的是人生。」(散文《公寓生活記趣》,《流言》1944)

## Don't / Over-mimic
- **Failure mode** (3 類 cliché): (1) 月亮 / 華美的袍 / 紅玫瑰白玫瑰 direct quote；(2)「蒼涼」「繁華」「淒美」抽象情緒名詞堆疊；(3) 瓊瑤腔 / 王家衛獨白腔的抒情長嘆，丟掉張的冷筆
- **Mitigation**: "借句型與比喻落點方式；禁月亮/華美的袍/紅白玫瑰直引；禁抽象情緒名詞堆疊；名詞句收束"
