## 2. Cluster A — 長文案框架（Agent 1）

### A.1 旧 PASONA（神田昌典 / 1999 通說，書籍正式化 2016）

**canonical**：神田昌典 2016『稼ぐ言葉の法則』(Diamond 社)

5 段階：
- **P**roblem（問題）— 読者の悩み明確化
- **A**gitation（扇動／煽り）— 放置結果の危機感刺激
- **So**lution（解決策）— 解決法 + 具体商品（合体）
- **N**arrow down（絞り込み）— 対象・限定性
- **A**ction（行動）— CTA

**claim → source mapping**：
- 1999 年提唱は通說（神田 1999『あなたの会社が 90 日で儲かる!』未正式記載）→ [canvas-works.jp](https://www.canvas-works.jp/journal/newpasona/) / [shanon.co.jp](https://www.shanon.co.jp/blog/entry/pasona_rule/)
- 2016 年が初正式文書化 → [kandamasanori.com/books/745](https://kandamasanori.com/books/745/)
- 5 段階定義 → 3 獨立 JP 來源一致（canvas-works / blastmail / shanon）

### A.2 新 PASONA（神田昌典 2016）

**canonical**：神田昌典 2016『稼ぐ言葉の法則 ——「新・PASONA の法則」と売れる公式 41』

6 段階（vs 旧）：
- P → **A**ffinity（親近感）— 共感・寄り添い（旧 Agitation を置換）
- **S**olution — 解決策（解決原理）
- **O**ffer（提案）— 価格・保証（旧 So の後半を独立させる）
- N, A — 同上

**旧→新變更**：① A 煽り→親近感、② So 合體 → S/O 分離。

**claim → source mapping**：
- 2016 年刊 → [kandamasanori.com/books/745](https://kandamasanori.com/books/745/)、[Amazon 4478067481](https://www.amazon.co.jp/dp/4478067481)
- Affinity 置換 Agitation → 5 獨立 JP 來源一致（innovation/canvas-works/shanon/satorifactory/swingroot）

### A.3 PASBECONA（神田昌典・衣田順一 2021）

**canonical**：神田昌典・衣田順一 2021『コピーライティング技術大全 —— 百年売れ続ける言葉の原則』(Diamond 社)

9 段階（在新 PASONA 中間插入 B/E/C）：
- P / A / S — 同上
- **B**enefit（利得）— 理想の未来
- **E**vidence（証拠）— 権威・実績
- **C**ontents（内容）— 商品詳細
- O / N / A — 同上

**claim → source mapping**：
- 2021 年刊行 → [kandamasanori.com/books/14787](https://kandamasanori.com/books/14787/)、[PR TIMES](https://prtimes.jp/main/html/rd/p/000000018.000040176.html)
- 9 段階名稱 → 3 獨立來源（almacreation/h9nfp/emomarke）一致
- B/E/C 歸屬：神田 + 衣田共著，切分不明

### A.4 QUEST（Michel Fortin 2005）

**canonical**：Michel Fortin 2005 部落格文章 "Want Better Copy? Go On A Quest!"

5 段階：Qualify / Understand / Educate / Stimulate / Transition

**claim → source mapping**：
- 2005 年提案 → [imnewswatch.com 2006/8 再掲](https://www.imnewswatch.com/2006/08/13/read-fortins-article-want-better-copy-go-on-a-quest/)、Fortin 本人 [michelfortin.com](https://michelfortin.com/blog/remember-these-5-copywriting-formulas/)
- Fortin 位置 QUEST 為 AIDA 補完發展型
- 「登山型」（上り=欲求 → 頂上=訴求 → 下り=行動）Fortin 自比喻

### A.5 PASTOR（Ray Edwards 2016）

**canonical**：Ray Edwards 2016 *How to Write Copy That Sells* (Morgan James Publishing)

6 段階（各字多義，英語原典為準）：
- **P** = Person / Problem / Pain（三義）
- **A** = Amplify and Aspirations
- **S** = Story / Solution / System（三義）
- **T** = **Transformation + Testimony**（兩義，日本移植常誤縮為 Testimony 單義）
- **O** = Offer（全體的 ~20% 以內）
- **R** = Response

**claim → source mapping**：
- 2016 年刊 → [Amazon 161448502X](https://www.amazon.com/Write-Copy-Sells-Step-Step/dp/161448502X)
- 日本語版 → [Amazon 4866220988](https://www.amazon.co.jp/dp/4866220988)
- T 兩義 → [Cooler Insights 2017](https://coolerinsights.com/2017/03/copywriting-tips-pastor/)、[Instacopy.ai](https://instacopy.ai/blog/pastor-copywriting-formula/)

## 3. Cluster B — 短文案傳統（Agent 2）

### B.1 AIDA / AIDMA

**canonical**：
- AIDA：E. St. Elmo Lewis 1898
- AIDMA：Samuel Roland Hall 1924『Retail Advertising and Selling』

5 段階（AIDMA）：Attention → Interest → Desire → **Memory** → Action

**短文案的角色**：短文僅負責 A + I 階段；Desire / Memory / Action 由 body copy / CTA / repetition 承擔。

**SNS 時代**：AISAS(2004 電通) → SIPS / ULSSAS(2015 ホットリンク) 部分替代。但 BtoB、TVCM、店頭販促仍主流。

### B.2 キャッチコピー 3 秒ルール

經驗則（非 formula）：「人は 3 秒で広告を読むか捨てるか判断する」

推奨構造：
- 1 秒目：キャッチ + メインビジュアルで興味喚起
- 2 秒目：一番伝えたい情報
- 3 秒目：次の行動誘発

現代 SNS 時代（TikTok/Reels/Shorts）縮至 1-3 秒。

### B.3 糸井重里 哲學（短文 voice 伝統）

- 核心：「普通の言葉を生き生きさせる」
- 代表作：「おいしい生活。」（西武百貨店 1982-83，7 字）、「不思議、大好き。」（西武 1981，7 字）
- 典型字數：5-10 字
- 特徵：**意図的な曖昧さで受け手に解釈余地**；Anglo 的 "Just Do It" 是行動促進型、糸井是状態提案型

### B.4 谷山雅計 What × How

**canonical**：谷山雅計 2007『広告コピーってこう書くんだ！読本』（宣伝会議）

⚠️ **Attribution drift 校正**：What×How **不是**「四象限矩陣」，原書核心是：
- 「**散らかす → 選ぶ → 磨く**」三階段發想流程
- 「**なんかいいよね禁止**」— 「なぜ良いか」を言語化する発想體質
- 「描写 vs 解決」：不僅描述商品，須找新用途/意義
- 31 の発想体質訓練法（ideation 中心）

用於 ideation + evaluation 兩用。

### B.5 TCC（東京コピーライターズクラブ）

- 1958 年「コピー十日会」→ 1962 年 TCC 改称
- 毎年 4 月 TCC 賞選抜，秋刊『コピー年鑑』
- 1963 年第 1 巻以降継続
- Hall of Fame（2003 設立）— 糸井、谷山、岩崎、眞木、小霜ら殿堂入り

短文案的 canon 參照源（非 framework）。

### B.6 岩崎俊一 人生観コピー

- 哲學：「コピーは、作るものではなく、見つけるもの」
- 代表作：「やがて、いのちに変わるもの。」（ミツカン 2004，12 字）、「想像力と数百円。」（新潮文庫 1984，9 字）
- 典型字數：10-15 字
- 特徵：季節感・有限性・いのち的仏教無常観

⚠️ **Attribution drift 校正**：リゲイン「24 時間戦えますか？」歸 **杉山恒太郎 + リゲイン PJ**（非岩崎俊一，常見誤植）

### B.7 眞木準 ダジャレ伝統

- 宣言：「ダジャレではなく、オシャレ」
- 代表作：「でっかいどぉ。北海道」（全日空 1970s-80s，10 字）
- 典型字數：5-15 字（音韻・リズム重視）
- 特徵：日本語の**同音異義（掛詞）**は和歌以来の伝統継承

### B.8 小霜和也 現代短文案論

- 著作：『欲しい ほしい ホシイ』(2009) / 『ここらで広告コピーの本当の話をします。』(2014)
- 核心：「ヒトの本能から広告を読み解く」
- **反 PASONA 的観点**：煽り（PASONA）を排し、本能ドライバー分析で置換
- 短文評価基準：「一行で関係を作れるか」
- Anglo 類似：Cialdini《Influence》、Kahneman / Damasio（進化心理学・behavioral economics）

### Cluster B 分界摘要

| 主題 | 適用 | Formula 形式 | ideation/eval |
|---|---|---|---|
| AIDMA | 兩用 (短文 = A+I) | 5 段階 | evaluation |
| 3 秒ルール | 短文専用 | 経験則 | evaluation |
| 糸井 | 短文専用 | 哲学 | voice ref |
| 谷山 散らかす→選ぶ→磨く | 兩用 | 3 階段 + 31 訓練 | 兩方 |
| TCC 年鑑 | メタ (canon) | 査読制 | evaluation |
| 岩崎 | 短文専用 | 哲学 | voice ref |
| 眞木 | 短文専用 | 音韻技法 | ideation |
| 小霜 | 兩用 | 本能分析 | 兩方 |

**字數黃金範圍**：日本語キャッチコピー **7-15 字**（糸井 7 / 岩崎 12 / 眞木 10 / Yahoo トピック 13）

## 4. Cluster C — 發想法（Agent 3）

### C.1 マンダラート / Mandal-Art

**canonical**：今泉浩晃 1987 提唱、1988『創造性を高めるマンダラート発想法』

⚠️ **Attribution drift 校正 1**：「8 方向標準分類」（生活情境／情感觸發／數字等）**非今泉 1987 原作**，是後人衍生。原作只規定 3×3 結構 + 聯想發散。

⚠️ **Attribution drift 校正 2**：大谷翔平 OW64 **不是**今泉 マンダラート，是**原田隆史の原田 Method**（源自松村寧雄 1979 マンダラチャート，商標公司均不同）。

**原始結構**：
- 單層：3×3 = 9 格（中心 + 8 聯想）
- 展開版：每個關聯字再為中心 → 81 格，產出 72 個關聯概念

**自動化邊界**：
- LLM 可做：單層 8 格聯想、展開 72 格、抗漂移審查
- LLM 不可做：「意外な観点」識別、從 72 格挑選有效 concept（需人類直覺）

### C.2 KJ法

**canonical**：川喜田二郎 1967『発想法』中公新書 136、1970『続・発想法』中公新書 210、1986『KJ法』（完整版）

**6 階段**（多數次級來源採 6 版本，原作 Wikipedia 列 4）：
1. テーマ設定 / ブレインストーミング
2. **ラベル作成**（一卡一概念）
3. **グループ編成**（感性親和性で聚類）
4. **表札づくり**（為每組命名）
5. **図解化（A 型）**（空間配置 + 因果）
6. **文章化（B 型）**（敘述文）

**定位**：専用於**収束**（非發散）；本身不產生新 idea，整合過程中湧現新洞察（「渾沌をして語らしめる」）。

**自動化邊界**：
- LLM 可做：卡片摘要、敘述化 B 型
- LLM 半自動（需人類 checkpoint）：群組化（embedding 偏離感性親和）、表札命名
- LLM 不可做：圖解 A 型（空間語言）

### C.3 谷山雅計 散らかす→選ぶ→磨く

**canonical**：谷山雅計 2007『広告コピーってこう書くんだ！読本』

⚠️ **Attribution drift 校正 3**：谷山「What×How 四象限」**不在原書**。原書核心結構：

1. **散らかす**（發散）— 大量出候選，不自我審查
2. **選ぶ**（收斂）— 以讀者視角選
3. **磨く**（精煉）— 字句打磨
4. 貫通的「なんかいいよね禁止」規則
5. 31 の発想体質訓練法

What / How 是廣告業界通用戰略概念（Creative Strategy），**非谷山首創**。

### C.4 Verbalized Sampling（補充）

**canonical**：Jiayi Zhang et al. 2025 "Verbalized Sampling: How to Mitigate Mode Collapse and Unlock LLM Diversity" ([arXiv:2510.01171](https://arxiv.org/abs/2510.01171))

技術：prompt LLM 輸出 probability distribution（例："Generate 5 jokes and their probabilities"）→ 2-3x diversity 提升。

- training-free, model-agnostic（GPT/Claude/Gemini/Llama）
- 正交 temperature
- 官方實作：[CHATS-lab/verbalized-sampling](https://github.com/CHATS-lab/verbalized-sampling)
