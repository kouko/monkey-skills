# Long-Form Copywriting — PASONA Canon

Tier 3 standard: fully self-contained. Consolidates the three PASONA-
family long-form frameworks originating in 神田昌典's canon —
旧 PASONA (5 段階), 新 PASONA (6 段階), PASBECONA (9 段階) — with
書籍首載年度、段階定義、適用字數範囲、段階間 flow 設計原則、および
ethics 邊界を定義する。本 standard は copywriting-team v1.0.0 の
三層分流（長／中／短）における「長文案層」の canon 參照源である。

## Critical Attribution Corrections

- **drift #4：PASONA「1999 年提唱」は通說**。
  通行説では「神田昌典が 1999 年にセミナー等で提唱し、同年の
  『あなたの会社が 90 日で儲かる!』で公表」とされるが、1999 年の
  同書に **PASONA の 5 段階を formula として明示記載した事実は
  確認できない**（次級來源 canvas-works / shanon の一致証言）。
  **canonical は 2016 年『稼ぐ言葉の法則』をもって書籍首載とする**。
  1999 年を「書籍初記載」として引用する文献は二次資料の伝聞で
  あり、PASONA の canon としては採用しない。

- **drift #5：PASBECONA 命名年は 2021 年であり、2016 年ではない**。
  2016 年『稼ぐ言葉の法則』は **新 PASONA (6 段階)** の書であり、
  PASBECONA の 9 段階構造（B/E/C 挿入）は **2021 年『コピー
  ライティング技術大全』** で初めて体系化された。2016 年を
  PASBECONA 首載年と記す引用は誤り。B/E/C 三段は神田 + 衣田順一
  共著のため、各段階の具体的 creditor（どちらの発案か）の切分は
  二次資料では確認できない — 共著と表記する。

- **drift 非対象だが関連注意：旧 PASONA の Agitation を「煽り」
  方向へ強化する解釈**は一部次級來源に存在するが、新 PASONA で
  神田自身が Affinity に置換した経緯から、煽り強化は 本 canon の
  推奨路線ではない（`persuasion-ethics.md` と cross 参照すべき
  問題であり、本 standard 内では「Anti-Patterns」節で言及する
  のみに留める）。

## Framing — 三框架の位置づけ

PASONA 系列は **単一公式ではなく、字數帯・媒体・説得厚に応じて
使い分ける 3 世代の framework family** である。旧 PASONA は短中文
（DM / メール）、新 PASONA は中長文（LP / メルマガ）、PASBECONA は
長文専用（Web LP / 高額商品）を射程とする。切り替え軸は以下の 3 つ。

- **説得の厚さ** — どれだけ中間説得 (Benefit + Evidence + Contents)
  が必要か
- **段階間の離脱リスク** — 字數が伸びるほど離脱点が増え、Affinity
  (共感) と Narrow down (絞り込み) の役割が重くなる
- **倫理的トーン** — Agitation (煽り) は短文案では許容され得るが、
  長文では Affinity (親近感) による共感起点のほうが 完読率 / LTV
  で優れると神田自身が 2016 年に述べている

## 旧 PASONA (5 段階) — 神田 2016 書籍首載

5 段階の **P - A - So - N - A** 構造。Agitation と Solution の境界が
緩く、「So」は Solution の先頭 2 文字を合体表記する旧式記法。

| 段階 | 名称 | 該段階の任務 (3-5 行) |
|---|---|---|
| **P** | Problem (問題) | 読者が抱えている悩み・課題を明確化する。読者自身がまだ言語化していない痛点を、読者の言葉で描写して「これは自分のことだ」と自覚させる。 |
| **A** | Agitation (扇動 / 煽り) | 放置した場合の悪化・損失・機会喪失を具体的に描き、危機感を喚起する。Problem を「今すぐ動かねばならない問題」へ格上げする段階。 |
| **So** | Solution (解決策) | 解決の原理（なぜ / どうやって解決するのか）と、具体的商品・サービスを **同一段で** 提示する合体表記。旧 PASONA の特徴的圧縮点。 |
| **N** | Narrow down (絞り込み) | 対象者・数量・期間・価格の限定性を提示する。「あなたのための解決策であり、今しか入手できない」の二重絞り込み。 |
| **A** | Action (行動) | 具体的 CTA（ボタン文言・電話番号・申込手順）を提示。迷わず 1 クリックで完結する設計。 |

**適用字數**：1,500〜5,000 字。
**適用媒体**：DM、メール、短中長の セールスレター、広告チラシ、
テレアポ trial のスクリプト。
**特徴**：So 合体により説得の密度が高い分、中間の共感・証拠は
薄い — 読者が既に問題認識を持つ前提の媒体に適する。

## 新 PASONA (6 段階) — 神田 2016『稼ぐ言葉の法則』

6 段階の **P - A - S - O - N - A** 構造。旧の A (Agitation) を
**Affinity (親近感)** に置換し、旧の So (合体) を **S (Solution) +
O (Offer)** に分離した 2 点が canonical な変更。

| 段階 | 名称 | 該段階の任務 (3-5 行) |
|---|---|---|
| **P** | Problem (問題) | 旧 PASONA と同様。読者の悩みを読者の言葉で描写し自覚させる。 |
| **A** | **Affinity (親近感 / 共感)** | 「私もあなたと同じように悩んだ」「その気持ち、わかります」と寄り添う段。旧 Agitation (煽り) を共感起点に置換したのが canonical 変更点 #1。 |
| **S** | Solution (解決策) | 解決の原理を提示する。「なぜこの方法で解決できるのか」の論理的説明。商品・サービスそのものの詳細には未だ踏み込まない。 |
| **O** | Offer (提案) | 商品・サービス・価格・保証・特典を具体的に提示する段。旧 So の後半を独立させたのが canonical 変更点 #2。 |
| **N** | Narrow down (絞り込み) | 対象・数量・期間の限定性。旧 PASONA と同じ。 |
| **A** | Action (行動) | 具体的 CTA。旧 PASONA と同じ。 |

**旧→新 canonical 変更の論点**：

- **変更 #1 (A: Agitation → Affinity)**：煽りによる短期 CVR 最適化は
  完読率と LTV を毀損するとの神田自身の反省を受け、共感起点に
  置換された。Affinity は「読者との並走」であり、焦燥感ではなく
  安心感を起点とする。
- **変更 #2 (So → S / O 分離)**：原理 (Solution) と 具体商品提示
  (Offer) を分離することで、原理への納得 → 商品への期待 の二段
  階跳躍が設計可能になり、中長文での説得 flow が滑らかになる。

**適用字數**：3,000〜10,000 字。
**適用媒体**：LP (ランディングページ)、メルマガ連載、白書、
資料請求トリガーの企画書。

## PASBECONA (9 段階) — 神田・衣田 2021『コピーライティング技術大全』

9 段階の **P - A - S - B - E - C - O - N - A** 構造。新 PASONA の
S (Solution) と O (Offer) の間に **B (Benefit) - E (Evidence) -
C (Contents)** の 3 段を挿入することで、長文案における説得厚を
canonical に確保する設計。

| 段階 | 名称 | 該段階の任務 (3-5 行) |
|---|---|---|
| **P** | Problem (問題) | 新 PASONA と同じ。読者の悩みを明確化。 |
| **A** | Affinity (親近感) | 新 PASONA と同じ。共感起点。 |
| **S** | Solution (解決策) | 解決の原理。新 PASONA と同じ。 |
| **B** | **Benefit (利得)** | 解決後に読者の生活がどう変わるか、理想の未来像を描く。「手に入る結果」であり、商品スペックではない。 |
| **E** | **Evidence (証拠)** | Benefit の根拠。権威 (専門家推薦)、実績 (売上・受賞)、顧客の声、統計データなど客観的裏付けを提示する段。 |
| **C** | **Contents (内容)** | 商品・サービスの詳細。構成要素、使い方、同梱物、サポート内容。スペック情報はこの段に集約される。 |
| **O** | Offer (提案) | 価格、保証、特典、申込条件。新 PASONA O の役割と同じ。 |
| **N** | Narrow down (絞り込み) | 対象・数量・期間の限定。 |
| **A** | Action (行動) | 具体的 CTA。 |

**B / E / C 挿入の論理**：

- **B (Benefit)** は Solution の抽象性を読者の生活像に着地させる。
  「解決できる」から「こうなれる」への 翻訳段。
- **E (Evidence)** は Benefit への疑念を先回りで解消する。長文
  では読者が「本当に？」と立ち止まる瞬間が必ず訪れるため、
  証拠を接続することで離脱を防ぐ。
- **C (Contents)** は B と E で醸成された期待を、商品の具体像に
  接地させる。ここで初めて スペック / 使用方法 が出現する。
- この 3 段の挿入順は **抽象 (Benefit) → 客観 (Evidence) → 具体
  (Contents)** であり、逆順は認知負荷を上げて離脱を招く。

**適用字數**：5,000〜20,000 字 以上。
**適用媒体**：Web LP (高額商品・コーチング・BtoB)、セールスレター、
セミナー勧誘ページ。

**B/E/C 歸屬の uncertainty**：神田 + 衣田順一 の共著であり、
各段階の creditor（どちらの発案か）の切分は二次資料では確認
できない。本 standard では **共著表記** とし、段階ごとの個別
帰属は避ける。

## 三框架の適用表（分流マトリクス）

| Framework | 段階數 | 字數 | 媒体例 | 説得厚 | トーン起点 |
|---|---|---|---|---|---|
| 旧 PASONA | 5 | 1,500–5,000 字 | DM, メール, チラシ | 薄 (So 合体) | Agitation (煽り) |
| 新 PASONA | 6 | 3,000–10,000 字 | LP, メルマガ, 白書 | 中 (S/O 分離) | Affinity (共感) |
| PASBECONA | 9 | 5,000–20,000+ 字 | 高額 LP, BtoB LP, セミナー勧誘 | 厚 (B/E/C 挿入) | Affinity (共感) |

**選定基準**：

- 目的の字數が 5,000 字を超える想定 → PASBECONA を default とする
- 3,000–5,000 字帯で Affinity 起点を維持したい → 新 PASONA
- 短中文（3,000 字未満）かつ既に問題認識を持つ読者 → 旧 PASONA
- トーンが煽り不許容（BtoB / 医療 / 金融）→ 旧 PASONA は非推奨

## 段階間 flow 設計原則

長文 PASONA 系列を設計するときの 4 つの canonical 原則。

- **離脱点の明示**。段階移行の境目（P→A, A→S, S→B など）は
  読者が最も離脱しやすい場所であり、接続詞・小見出し・問いかけ
  で「次の段階への橋渡し」を明示する。特に PASBECONA の B→E
  境界は期待を証拠で受ける重要 transition。
- **字數比例**。旧 PASONA の場合 P:A:So:N:A ≒ 2:2:3:2:1 が
  canonical な経験則。新 / PASBECONA では 段階増加に応じて
  O と E の比重が増え、5,000 字の PASBECONA なら P:A:S:B:E:C:O:N:A ≒
  1:1:1:1:2:2:1:0.5:0.5 が目安。
- **Affinity の厚さ基準**。新 / PASBECONA での Affinity は、
  Problem と同等以上の分量を割くのが canonical。薄い Affinity は
  「共感風の台詞」になり、Problem との区別が失われる。
- **Narrow down と Action の近接**。N と A は必ず隣接させ、
  限定性を認識した直後に CTA を置く。間に別段階を挟むと緊急性が
  失われる。

## ethics 境界（`persuasion-ethics.md` 管轄）

Agitation 段階での煽り過度（不安商法、恐怖訴求の過剰使用）は
本 canon の範囲外であり、**`persuasion-ethics.md`** で管轄する。
本 standard では「旧 PASONA の Agitation は短中文で許容され得る
が、長文での Affinity 置換が神田自身の反省を経た canon である」
と記すのみに留める。景品表示法・薬機法・特定商取引法 に関わる
表現制限も同様に ethics 側で扱う。

## Anti-Patterns

- **1999 年を書籍首載として引用する**。通說であり、2016 年
  『稼ぐ言葉の法則』が canonical 首載年である (drift #4)。
- **PASBECONA を 2016 年命名とする**。2021 年『コピーライティング
  技術大全』が canonical 命名年である (drift #5)。2016 年は
  新 PASONA (6 段階) の年。
- **煽り式 Agitation を Affinity と呼ぶ**。Affinity は 共感・並走
  であり、Agitation の柔らかい言い換えではない。「不安を感じて
  いませんか？」で終わる文は Agitation 寄り。「私も同じでした」
  で始まる文が Affinity。
- **Offer 段を省略して Solution から直接 Action へ**。新 / PASBECONA
  で Offer を省略すると、価格・保証が不透明なまま CTA となり
  CVR が落ちる。旧 PASONA では So が Offer を兼ねるが、新以降は
  独立必須。
- **PASBECONA で Contents を Benefit の前に置く**。抽象→客観→具体
  の順序が canonical であり、逆順は認知負荷を上げる。スペック
  先出しは新 PASONA 以降では Anti-Pattern。
- **B/E/C 段階の creditor を神田 or 衣田 単独と断定する**。共著で
  あり切分は二次資料で確認できない。
