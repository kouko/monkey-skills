# Persuasion Psychology Anchor 說服心理學基石

Tier 2 common-layer standard：為 copywriting-team 的 JP 框架（PASONA /
PASBECONA / BEAF / キャッチコピー）提供**西方心理學 anchor**。JP 框架
在結構層面已 canonical 化，但其說服機制若無心理學解釋則易流於 cookbook 套用。
本 standard 列出 Cialdini 6 原則、Schwartz 5 awareness levels、Kahneman
System 1/2、Thaler & Sunstein 的 Nudge 共 4 套心理學 framework，作為
copywriting 的理論基石。本檔只規範「如何利用心理學機制說服」；倫理・
法律邊界（FTC / 景品表示法 / dark pattern / 小霜「嘘をつかない」）由
copywriting-team 的倫理 standard 獨立規範，不在本檔範圍。

## Primary Sources

- Robert Cialdini (1984/2006) *Influence: The Psychology of Persuasion*, Harper Business — 6 原則 (reciprocity / commitment & consistency / social proof / authority / liking / scarcity) の canonical 出典。1984 年初版の全面改訂版が 2006 年 Collins Business。7 つ目の Unity 原則は 2016 *Pre-Suasion* で追加。
- Eugene Schwartz (1966) *Breakthrough Advertising*, Boardroom/Bottom Line Personal. 2017 年復刻版 Titans Marketing — 長文案 LP の canonical 出典。「5 levels of awareness」「3 stages of market sophistication」の概念元祖。Schwartz は Dan Kennedy / Gary Halbert / David Ogilvy とともに 20 世紀 direct-response copywriting の 4 巨頭。
- Daniel Kahneman (2011) *Thinking, Fast and Slow*, Farrar, Straus and Giroux — System 1 (高速・直感・感情的) / System 2 (低速・熟慮・論理的) の dual-process theory；prospect theory と loss aversion の popularized 出典。Kahneman は 2002 年ノーベル経済学賞受賞。
- Richard Thaler & Cass Sunstein (2008) *Nudge: Improving Decisions About Health, Wealth, and Happiness*, Yale University Press — choice architecture と default option、nudge の canonical 出典。Thaler は 2017 年ノーベル経済学賞受賞。2021 年に *Nudge: The Final Edition* が刊行。

## Cialdini 6 原則 + copywriting 應用

Cialdini 1984/2006 的 6 原則按 canonical 順序。各項都是**合法工具**；
操控化後果（false scarcity / fake testimonial 等）屬 copywriting-team の
倫理 standard scope、本檔不再重複。

| # | 原則 | 定義（Cialdini canonical） | Copywriting 典型応用 | PASONA / PASBECONA anchor |
|---|------|-----------------------------|---------------------|---------------------------|
| 1 | Reciprocity（互恵性） | 無料で受け取ると返報の社会的 pressure が生じる | 無料試用、無料 PDF レポート、無料 webinar、lead magnet → main offer の段階設計 | Benefit / Offer 段：「既にこれだけ提供した」メタ互恵 |
| 2 | Commitment & Consistency | 小さな commitment を得ると後続のより大きな commitment に応じやすい | 段階的 CTA（メアド→trial→有料）、yes-ladder、セルフイメージ訴求 | Problem / Affinity 段：「あなたは X で困っている」同意が後続 Offer を予約 |
| 3 | Social Proof（社会的証明） | 不確実な状況で多数派の行動を模倣 | Testimonial、「X 人利用中」、case study、レビュー数・星評価 | Evidence 段の主要 building block。代表性は FTC §255.2（倫理 standard 参照） |
| 4 | Authority（権威） | 専門家・権威者の推奨に従う | 専門家 endorsement、credentials、科学的証拠・論文引用 | Evidence 段の social proof と並ぶ 2 本柱。虚偽権威は FTC §255.1 違反 |
| 5 | Liking（好感） | 類似性・親近感・外見的魅力が説得力を高める | 読者セグメントの自己投影 narrative、brand persona、founder story | Affinity 段（新 PASONA 2016）の心理学 anchor。糸井・岩崎 voice が JP 実装形 |
| 6 | Scarcity（希少性） | 限定・残少・期限で価値知覚が上昇 | 限定数量、期限、限定アクセス | **Narrow down 段**の anchor。**真実性必須**（虚偽は景品表示法 有利誤認 — 倫理 standard §False scarcity 参照） |

Cialdini 2016 *Pre-Suasion* で 7 つ目の Unity（共同体的一体感）が追加されたが、
v1.0.0 では上記 6 原則のみを scope とする。

## Schwartz 5 Levels of Awareness（長文 LP 必備）

Schwartz 1966 *Breakthrough Advertising* 第 1 部で定義。読者の「問題意識・
商品知識」段階が長文案の構造選択と copy density を決める。長文 LP
（PASBECONA レンジ）では**読者がどの awareness level にいるか**を最初に
診断する必要がある。

| Level | 読者状態 | 必要な copy | 適合フレーム（v1.0.0 scope） |
|-------|---------|-------------|-----------------------------|
| **1. Most Aware** | 商品・価格・特徴を既知 | 価格・オファー・ボーナス・即時性 | 旧 PASONA の Narrow down + Action 圧縮、短い promotional email |
| **2. Product Aware** | カテゴリ既知、複数選択肢を比較中 | 差別化、比較優位、証拠、保証 | BEAF（Advantage 段が核）、新 PASONA の Offer 段 |
| **3. Solution Aware** | 解決策の存在を知るが具体商品を知らない | 自社ソリューションが優れる理由の教育 | 新 PASONA の Solution 段、PASBECONA の Solution + Benefit |
| **4. Problem Aware** | 痛みは自覚、解決手段を知らない | 問題を言語化・共感、解決策の可能性を示唆 | 新 PASONA の Problem + Affinity、PASBECONA の Problem + Affinity + Benefit |
| **5. Unaware** | 問題すら自覚していない | ストーリー・比喩・事例で問題を「発見」させる | PASBECONA の Problem + Affinity + Benefit + Evidence（フル長文） |

**core rule (Schwartz 1966)**：**Unaware 読者に直接 Offer を closer として
使うのは禁止**。awareness ladder を飛ばすと conversion は崩壊する。Level 5
には narrative による problem discovery が必須。

## Kahneman System 1 / System 2 — 短文 vs 長文の使い分け

Kahneman 2011 の dual-process theory を copywriting に適用：

### System 1（高速・直感・感情的）

- 瞬間判断、熟慮しない、感情とヒューリスティックに駆動
- **適合する copy**：短文案（キャッチコピー・見出し・SNS）
- **使う武器**：情緒、リズム、familiarity、ビジュアル連動、7–15 字
  の身体的インパクト
- **JP 伝統との対応**：糸井「状態提案」、岩崎「いのち系」、3 秒ルール
  の初秒判断は System 1 領域

### System 2（低速・熟慮・論理的）

- 論理検証、証拠評価、比較検討
- **適合する copy**：長文案（PASBECONA 相当レンジ / sales letter）
- **使う武器**：論点構成、証拠、論理的一貫性、数字、Case study
- **JP 伝統との対応**：神田 PASONA の Solution-Offer シーケンスは
  System 2 ルートで説得

### Implication

**同一キャンペーン内での分業**：短文案で System 1 を掴む → body で
System 2 を納得させる → closing で再び System 1 の情緒に戻る。
Kahneman の insight は「どちらか一方では売れない」こと：System 1 が
反応しなければ body を読まず、System 2 が納得しなければ購入しない。

## Nudge（Thaler & Sunstein 2008）— Choice architecture

Thaler & Sunstein 2008 の nudge 概念：選択肢の提示方法（choice
architecture）が決定に大きく影響する。copywriting との関連：

- **Default option の力**：opt-in vs opt-out のデフォルト設定が購読率・
  加入率を劇的に変える。copy 層では「デフォルトで選ばれている選択肢」
  の提示順・推奨マーカー（"Most popular" 等）の設計。
- **Anchoring（係留効果）**：比較参照点の提示（定価表示 → 割引価格）。
  景品表示法 有利誤認の違反リスクがあるため真実の anchor 必須（倫理
  standard §有利誤認 参照）。
- **Framing**：同じ事実を「90% 生存率」と「10% 死亡率」で提示すると
  決定が変わる（Kahneman 1979 prospect theory 応用）。copywriting の
  語彙選択に直接作用。

Nudge は Cialdini 6 原則と相補：Cialdini は**説得のドライバー**、Nudge
は**選択環境の設計**。両者は倫理的 libertarian paternalism の範囲内で
使用する限り合法；操控化すると dark pattern（倫理 standard 参照）。

## PASONA フレームワークの心理学 anchor マッピング

新 PASONA (神田 2016) / PASBECONA (神田・衣田 2021) の各段階に対する
西方心理学 anchor の対応表：

| PASONA 段階 | 主要心理学 anchor | 補助 anchor |
|------------|------------------|-------------|
| Problem | Schwartz Level 4→3 移動 | Kahneman System 1 (痛みの瞬間喚起) |
| Affinity | Cialdini Liking | 糸井 / 岩崎 JP 情緒 voice |
| Solution | Schwartz Level 3→2 移動 | System 2 (論理的妥当性) |
| Benefit (PASBECONA) | Kahneman prospect theory (gain frame) | Cialdini Authority（専門家が benefit を保証） |
| Evidence (PASBECONA) | Cialdini Social Proof + Authority | System 2 (証拠の論理検証) |
| Contents (PASBECONA) | Schwartz Level 2→1 移動 | System 2 (特徴比較) |
| Offer | Kahneman loss aversion (失うもののフレーム) | Cialdini Reciprocity（特典の互恵） |
| Narrowing | Cialdini Scarcity | **真実性必須** — 倫理 standard §景品表示法 参照 |
| Action | Cialdini Commitment & Consistency | Nudge default / recommended marker |

このマッピングは「PASONA がなぜ効くか」を心理学で説明する道具であり、
**逆方向（心理学から PASONA を機械的に導出する）** には使えない。

## Anti-Patterns

- **Unaware 読者に直接 Offer closer を使う**。Schwartz の awareness
  ladder を飛ばすと conversion は崩壊。Level 5 には narrative + problem
  discovery が必要。
- **短文にも System 2 要素を詰め込む**。7–15 字の短文案で数字・証拠・
  論理を並べると System 1 ルートが機能せず、読者は 3 秒で離脱。
- **長文を全部 System 1 の情緒だけで書く**。高額商品・BtoB の PASBECONA
  レベル長文で感情のみ（Agitation 重視）では System 2 が納得せず、
  conversion が bounce。
- **Cialdini 6 原則を cookbook で機械適用**。文脈・ブランド voice・
  読者 awareness を無視して「scarcity だから残り 3 名」を乱用すると
  信頼毀損（+ 景品表示法 リスク — 倫理 standard 参照）。
- **Schwartz level 診断なしに framework を選ぶ**。どの PASONA 変種
  （旧 / 新 / PASBECONA）を使うべきかは awareness level と product
  complexity の掛け算で決まる（Level 5 + 高額 → PASBECONA / Level 1 +
  既知 → 短文）。
- **Anchor や framing を真実から外す**。Nudge の合法性は libertarian
  paternalism（選択肢自体は変えない）に依存；虚偽 anchor は景品表示法
  有利誤認。
- **PASONA の Affinity 段で liking を演じる**が brand voice と整合
  しない。brand voice guide との整合性を確認する（voice standard 参照）。
- **Loss aversion を脅迫に推し進める**。「失う」フレームは System 1
  で強いが、扇動過剰は小霜 2014「嘘をつかない」+ PASONA 反煽り論点に
  抵触；倫理境界は倫理 standard が規範。
