# A1 — Past Application (full)

Source: Sherwood, *Seeing the Forest for the Trees*, Chapter 10. All
three applications below are workshop-typology observations from the
same chapter — not independently corroborated case studies. Treat as
calibration vignettes for the framing-vs-analysis split, not as
empirical evidence for the quadrant's predictive validity. See the
SKILL.md Boundary section for the full V1-weak disclosure
(anecdotal evidence base, prior-art overlap with DiSC / MBTI / Hogan /
Situational Leadership, and the circular Guides-quadrant definition).

## 1. Scenario-planning asymmetric reception (Ch 10)

Sherwood's explicit application: a leader's quadrant predicts whether
they will use or discard a scenario-planning output.

- **Gods** reject it as "in the way" of decisiveness. The plural
  framing reads as indecision; the executive demands a single
  recommendation extracted from the work and discards the rest.
- **Grinders** embrace it as the latest method but want a single
  recommended answer extracted from it. They treat the method as a
  delivery mechanism for an answer, not as an exploration.
- **Gamblers** want odds attached to each scenario, then collapse the
  plural framing into an expected-value calculation. The scenarios
  become inputs to a probability-weighted decision, not a
  decision-making frame in themselves.
- **Guides** engage with the plural framing directly — the native
  audience for the artefact as-is.

**Framing-vs-analysis use:** the underlying scenario set is the
*same* in all four cases. What changes is the presentation layer —
recommendation-with-fallbacks (Gods), framework wrapper (Grinders),
expected-value table (Gamblers), or native exploration (Guides).
The analysis layer is the anchor; the framing layer is the variable.

## 2. Consulting-fad acceptance pattern (Ch 10)

Sherwood links the Grinder quadrant to the recurring observation
that some executives "always pay huge sums for the latest business
school fad" — the quadrant explains why the same exec hires the same
kind of method repeatedly across BPR, Six Sigma, Balanced Scorecard,
OKRs, and so on.

The mechanism: a Grinder believes the future is predictable and that
the right method extracts the answer. Each new method is procured
under that belief; the belief is not falsified by previous methods'
failure because the failures are attributed to inadequate execution,
not to the predictability assumption.

**Framing-vs-analysis use:** when presenting any analysis to a
Grinder-quadrant executive, expect them to ask "what is the right
answer the method tells us?" The framing adaptation is to wrap the
analysis in a structured-process container (e.g., a 5-step framework
or a maturity model) so the executive can engage with the surface
form — without changing the analysis underneath.

## 3. Method-rejection by Gods-style leadership

Sherwood positions the Gods quadrant as structurally hostile to any
framework that surfaces uncertainty. The leader has organized her
self-concept around being able to predict, so a method that
foregrounds plural futures threatens that self-concept and gets
rejected as method-politics, not on its merits. The book provides
this as a workshop-typology observation, not an empirically-
validated finding.

**Framing-vs-analysis use:** when presenting to a Gods-quadrant
executive, lead with a decisive recommendation and treat the
scenarios as fallback branches — "if X, we do Y; if not-X, we do Z."
The analysis (the scenarios + the lever map) is unchanged; only the
surface is restyled to honor the executive's predict-and-control
self-concept. Do not strip the scenarios out — that is analysis
change, not framing change, and it discards the audit trail.

## Cross-cutting note — the framing-vs-analysis test

In all three cases above, the framing change is legitimate (surface
form differs; underlying scenarios + levers unchanged) and the
analysis change would be illegitimate (dropping inconvenient
scenarios to please an executive). The Boundary section of SKILL.md
codifies this as a NOT-use clause: framing adaptation must not
smuggle in a different analysis.
