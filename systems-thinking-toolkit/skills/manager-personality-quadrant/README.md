[English](README.md) | [日本語](README.ja.md) | [繁體中文](README.zh-TW.md)

## Summary

⚠ V1-weak auxiliary — adapt strategy artifacts to executive personality via the framing-vs-analysis split. 2×2 (Gods / Gamblers / Grinders / Guides) is supporting vocabulary.

## When to use

- Presenting a strategy or scenario-planning artifact to an executive with a known style.
- CEO rejected a previous deck despite sound analysis, and the team wants to fix framing rather than redo analysis.

## How to invoke

```
/systems-thinking-toolkit:quadrant
```

## What you get

- A framing-adapted artifact targeted at the executive's style — analysis layer unchanged.
- A quadrant-aware presentation script (decisive recommendation / expected-value table / 5-step framework / plural-futures exploration).

## ⚠ Boundaries

- **V1-weak per Stage 1.5** — evidence base is anecdotal; Sherwood cites no empirical leadership-style research.
- **Stronger empirically-grounded alternatives exist**: DiSC, MBTI, Hogan Assessments, Situational Leadership (Hersey–Blanchard) all carry peer-reviewed validity literature this 2×2 does not.
- **Guides-quadrant has a circular-definition warning in the source** — Guides are defined as those who use scenario planning, then said to benefit from it.
- Do NOT use for hiring, promotion, performance management, or psychometric assessment.
