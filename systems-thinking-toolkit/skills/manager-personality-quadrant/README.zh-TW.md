## 摘要

⚠ V1-weak 輔助技能 — 透過 framing-vs-analysis split，將策略產出依高階主管的風格調整呈現方式。2×2（Gods / Gamblers / Grinders / Guides）為輔助詞彙。

## 何時使用

- 要向已知風格的高階主管呈現策略或情境規劃產出時。
- 分析本身合理但 CEO 卻否決了前一份簡報，團隊想修正呈現框架而非重做分析。

## 如何啟用

```
/systems-thinking-toolkit:quadrant
```

## 你會得到什麼

- 依目標高階主管風格調整呈現框架的產出 — 分析層保持不變。
- 帶有 quadrant 意識的簡報腳本（決定性建議／期望值表／五步驟框架／plural-futures 探索）。

## ⚠ Boundary

- **Stage 1.5 判定為 V1-weak** — 證據基礎屬軼事性，Sherwood 並未引用實證的領導風格研究。
- **存在實證基礎更強的替代方案**：DiSC、MBTI、Hogan Assessments、Situational Leadership（Hersey–Blanchard）皆有此 2×2 所欠缺的同儕審查效度文獻。
- **Guides quadrant 在原書中存在循環定義警告** — Guides 被定義為「使用情境規劃者」，再被宣稱「從情境規劃中受益」。
- 不得用於招募、晉升、績效管理或心理測量評估。
