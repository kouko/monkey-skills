# manager-personality-quadrant

[English](README.md) | [日本語](README.ja.md) | [繁體中文](README.zh-TW.md)

## 概要

⚠ V1-weak 補助スキル — framing-vs-analysis split によって、戦略アーティファクトを経営者のスタイルに合わせて適応させる。2×2（Gods / Gamblers / Grinders / Guides）は補助的な語彙。

## こんなときに使う

- 既知のスタイルを持つ経営者に対して、戦略やシナリオプランニングのアーティファクトを提示するとき。
- 分析は妥当だったのに CEO が前回のデッキを却下し、分析をやり直すのではなく提示フレームを修正したいとき。

## 得られるもの

- 経営者のスタイルに合わせて提示フレームを適応させたアーティファクト — 分析レイヤーは変更しない。
- quadrant を意識したプレゼンテーションスクリプト（決定的な推奨／期待値テーブル／5 ステップフレームワーク／plural-futures 探索）。

## ⚠ Boundary

- **Stage 1.5 で V1-weak 判定** — エビデンス基盤は逸話的で、Sherwood は実証的なリーダーシップスタイル研究を引用していない。
- **実証的に裏付けの強い代替手段が存在する**：DiSC、MBTI、Hogan Assessments、Situational Leadership（Hersey–Blanchard）は、この 2×2 が持たない peer-reviewed の妥当性文献を備えている。
- **Guides quadrant にはソース上の循環定義警告がある** — Guides は「シナリオプランニングを使う者」と定義されたうえで「シナリオプランニングが有効」と語られる。
- 採用・昇進・人事評価・心理測定アセスメントには使用しないこと。
