# innovaction-martian-test

> 注意:V1-weak 補助スキル — 通常は代替未来を生成する際に
> `strategy-lever-and-cascade` ワークフロー内部から到達する。
> Martian テストによる特徴列挙 + 一特徴の摂動。

## こんなときに使う

- sk08 型シナリオプランニング内部での「自明な変種」問題(「同じ世界、
  GDP が 5% 高いだけ」)— Martian テストの密度基準が、摂動を生産的に
  するための運用上の完全性しきい値となる。
- 既存の TRIZ 実践がない場合の摂動法アイデア出し — 確立された系統的
  イノベーション枠組みを持たないチームの素早い導入として有用。

## 起動方法

```
/systems-thinking-toolkit:martian-test
```

## 得られるもの

- Martian テスト密度の特徴リスト(文脈ゼロの外部者が隣接物との区別に
  使える密度で 20〜40 項目)。
- 摂動済み特徴シナリオ(1 パスにつき 1 特徴を反転、3〜5 の下流結果と
  レバー設定への含意付き)。

## 注意:境界

- **Stage 1.5 により V1-weak** — Sherwood ブランド、単一章のみ、
  代替手法との独立査読比較なし。
- **TRIZ**(Genrich Altshuller, 1946+)および**形態分析**(Fritz Zwicky,
  〜1947)は経験的に裏付けられた、より強力な代替手法であり、産業 /
  航空宇宙 / フォーサイトで数十年の実績を持つ。
- チームが既に TRIZ または形態分析を実践している場合はそちらを使う
  こと — このスキルはより弱いツールで重複する。
