# innovaction-martian-test

> 警告:V1-weak 輔助技能 — 通常於產生替代未來時,從
> `strategy-lever-and-cascade` 工作流內部到達。Martian 測試特徵列舉
> + 單一特徵擾動。

## 何時使用

- sk08 型情境規劃內部的「瑣碎變體」問題(「同一個世界,GDP 高 5%」)
  — Martian 測試的密度門檻是讓擾動具生產力的運作完整性標準。
- 在團隊尚未實踐 TRIZ 時的擾動法構想產出 — 適合作為尚未建立系統性
  創新框架的團隊的快速入門。

## 啟動方式

```
/systems-thinking-toolkit:martian-test
```

## 你會得到什麼

- Martian 測試密度的特徵清單(密度足以讓零脈絡的外部者區分此物與
  鄰近物,20〜40 條)。
- 經擾動的特徵情境(每次擾動翻轉一個特徵,附 3〜5 個下游後果與
  槓桿設定意涵)。

## 警告:界線

- **依 Stage 1.5 判定為 V1-weak** — Sherwood 品牌、單章涵蓋、缺乏
  與替代手法之獨立同儕審查比較。
- **TRIZ**(Genrich Altshuller, 1946+)與**形態分析**(Fritz Zwicky,
  約 1947)是經驗證據更紮實的替代方案,在產業 / 航太 / 前瞻分析有
  數十年實證紀錄。
- 若團隊已實踐 TRIZ 或形態分析,請使用該等手法 — 本技能僅以較弱
  工具重複其功能。

## 技能定義

完整 R-I-A1-A2-E-B 本體、執行步驟、相關技能與稽核中繼資料,請見
[`SKILL.md`](SKILL.md)。
