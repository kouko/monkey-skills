## Step-0 router quick reference

- **Per-cycle delta shrinking toward asymptote** → Branch L
- **Per-cycle delta alternating sign, amplitude > noise band** → Branch V
- **Both visible** → run V first IF amplitudes comparable; if
  deceleration ≫ oscillation × 3, run L primarily and treat V as note.
- **Monotone divergence** (delta growing) → NOT this skill; return to
  cld-craft for R-loop intervention.
- **Looks like B-loop but target never updates from output** → Case B5
  pseudo-target dangle; intervention = question the belief, not relieve
  brake / do nothing.

## Loading rule

For a clean single-archetype case (signature unambiguous, single
B-loop or single R+B coupling), this index + the one matching case
section is sufficient. Read `cases.md` end-to-end when:

- Both archetypes are simultaneously active (run V/T/A before L per Step 0)
- The diagnosed brake is itself dynamic (e.g. growing brake, or
  saturating brake) — Case A1 + A2 interactions matter
- Recommending "do nothing" as the intervention — Case B3 sets the
  bar for when committee-style inaction is the correct call
