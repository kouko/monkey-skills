## Task 3 — Write sheets-confirm-write.sh (L1 append / L2 clear/overwrite)

- **Description**: Create `gws-toolkit/scripts/gws/sheets-confirm-write.sh` modeled on `safe-delete.sh`. Op-aware tier dispatch: `--op append` → L1, `--op clear` / `--op update` (range overwrite) → L2.
