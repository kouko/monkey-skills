# Ikigai Skill

[English](README.md) | [日本語](README.ja.md) | **繁體中文**

透過 4 軸 Ikigai framework 分析任何專案、產品或職涯，
找出其目的並診斷缺口。

## 四個軸

| Axis | 日文 | 含義 | 關鍵提問 |
|------|------|------|---------|
| 1 | 好きなこと | 你所熱愛的 | 是否激起真正的熱情？ |
| 2 | 得意なこと | 你所擅長的 | 你具備執行所需的技能嗎？ |
| 3 | 世界が求めること | 世界所需要的 | 是否有真實的需求？ |
| 4 | 報酬を得られること | 你能藉以獲得報酬的 | 在財務上是否可行？ |

## Method Type

Framework-driven 分析（不像 Socratic method 是 dialogue-driven）。
agent 依序帶領使用者走過每個軸，再診斷交集點或指出缺失的軸。

關鍵原則：Ikigai 存在於**所有四個**軸的交集。
當一個或多個軸缺失時，會出現特定的徵兆。

## 缺口診斷

| 缺失的軸 | 結果 | 徵兆 |
|----------|------|------|
| 你所熱愛的 | 空虛 | 賺錢且擅長，但無滿足感 |
| 你所擅長的 | 無力 | 有熱情卻無法交付成果 |
| 世界所需要的 | 孤立 | 擅長且有熱情，但沒人在乎 |
| 你能藉以獲得報酬的 | 不穩定 | 被熱愛且被需要，但財務不可持續 |

## 適用領域

| Domain | Axis 1（熱愛） | Axis 2（擅長） | Axis 3（需要） | Axis 4（付費） |
|--------|---------------|-----------------|----------------|--------------|
| 創業 | 創辦人熱情 | 團隊能力 | 市場需求 | 營收模式 |
| 職涯 | 每日動機 | 專業強項 | 產業需求 | 薪資 / 報酬 |
| 產品 | 使用者喜悅 | 技術可行性 | 使用者痛點 | 商業可行性 |
| 副業 | 週末動機 | 既有技能 | 受眾規模 | 變現路徑 |

## 文化起源

Ikigai 源於日本文化，大致可譯為「a reason for being」（生存的理由）。
這個概念因 Dan Buettner 對沖繩長壽的研究而引起國際關注。
4 軸 Venn diagram 模型雖屬西方詮釋，
卻提供一個對齊熱情、技能、需求與可行性的實用 framework。
