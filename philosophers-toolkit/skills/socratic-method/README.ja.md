# Socratic Method Skill

[English](README.md) | **日本語** | [繁體中文](README.zh-TW.md)

Socratic method に基づく dialogue-driven な指導 framework。

## このスキルの内容

### 古典的な Socratic 手法

| Method | 説明 | SKILL.md 内の位置 |
|--------|-----|---------------------|
| Elenchus（吟味） | 問いかけで矛盾を露呈する | State B |
| Maieutics（助産術） | ユーザー自身の考えを「産む」のを助ける | Core Philosophy |
| Dialectic（対話） | 対話を通して explore → 深化 → synthesize | States A/D/E |

### 現代的な応用

| Mode | 目的 | 技法 |
|------|------|------|
| Pedagogical（既定） | 概念理解を深める | 定義 → 反例 → 再構成 |
| CBT（療法的） | 認知再構成、否定的信念への挑戦 | 証拠の test → 代替解釈 |
| Legal/Analytical | 論理的推論、エッジケース分析 | 仮定的シナリオ、reductio ad absurdum |
| Coaching/Strategic | 意思決定、根本原因分析 | 結果分析、前提の探索 |

### 問いの分類

| 種類 | 目的 |
|------|------|
| Clarification | 曖昧な語を運用可能にする |
| Assumption Probes | 表に出ていない信念を掘り出す |
| Evidence | データと推論を求める |
| Perspective | 認知の柔軟性を促す |
| Implication | 論理を結論まで追う |
| Meta-Questions | 対話そのものを省みる |

### 安全措置

| 措置 | 説明 |
|---------|------|
| 心理的安全 | 苛立ち閾値、distress 時の mode 切替 |
| 倫理的境界 | 操作なし、思想誘導なし |
| 弁証法的バランス | 常に thesis と antithesis の両方を explore |
