# Socratic Method Skill

**English** | [日本語](README.ja.md) | [繁體中文](README.zh-TW.md)

A dialogue-driven guidance framework built on the Socratic method.

## What This Skill Contains

### Classical Socratic Methods

| Method | Description | Location in SKILL.md |
|--------|-------------|---------------------|
| Elenchus (cross-examination) | Expose contradictions through questioning | State B |
| Maieutics (midwifery) | Help users "give birth" to their own ideas | Core Philosophy |
| Dialectic (dialogue) | Explore → deepen → synthesize through conversation | States A/D/E |

### Modern Applications

| Mode | Purpose | Technique |
|------|---------|-----------|
| Pedagogical (default) | Deepen conceptual understanding | Definition → Counterexample → Reconstruction |
| CBT (therapeutic) | Cognitive restructuring, challenge negative beliefs | Evidence testing → Alternative explanations |
| Legal/Analytical | Logical reasoning, edge-case analysis | Hypothetical scenarios, reductio ad absurdum |
| Coaching/Strategic | Decision-making, root cause analysis | Consequence analysis, assumption probing |

### Question Taxonomy

| Type | Purpose |
|------|---------|
| Clarification | Operationalize vague terms |
| Assumption Probes | Uncover unstated beliefs |
| Evidence | Demand data and reasoning |
| Perspective | Encourage cognitive flexibility |
| Implication | Follow logic to its conclusion |
| Meta-Questions | Reflect on the dialogue itself |

### Safety Measures

| Measure | Description |
|---------|-------------|
| Psychological safety | Frustration threshold, mode switching on distress |
| Ethical boundaries | No manipulation, no ideological steering |
| Dialectical balance | Always explore both thesis and antithesis |
