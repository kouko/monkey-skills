# Socratic Method Skill

[English](README.md) | [日本語](README.ja.md) | **繁體中文**

建立於 Socratic method 之上的 dialogue-driven 引導 framework。

## Skill 內容

### 古典 Socratic 方法

| Method | 描述 | 在 SKILL.md 的位置 |
|--------|-----|---------------------|
| Elenchus（詰問） | 透過提問揭露矛盾 | State B |
| Maieutics（助產術） | 幫使用者「生出」自己的想法 | Core Philosophy |
| Dialectic（對話） | 透過對話 explore → 深化 → synthesize | States A/D/E |

### 現代應用

| Mode | 目的 | 技法 |
|------|------|------|
| Pedagogical（預設） | 深化概念性理解 | 定義 → 反例 → 重構 |
| CBT（治療性） | 認知重構，挑戰負面信念 | 證據檢驗 → 替代解釋 |
| Legal/Analytical | 邏輯推理、邊界案例分析 | 假設情境、reductio ad absurdum |
| Coaching/Strategic | 決策制定、根因分析 | 後果分析、假設探查 |

### 提問分類

| 類型 | 目的 |
|------|------|
| Clarification | 將模糊用語具體化 |
| Assumption Probes | 揭露未明說的信念 |
| Evidence | 要求資料與推理 |
| Perspective | 鼓勵認知彈性 |
| Implication | 將邏輯推到底 |
| Meta-Questions | 反思 dialogue 本身 |

### 安全措施

| 措施 | 描述 |
|---------|------|
| 心理安全 | 挫折臨界值，遇到 distress 時切換 mode |
| 倫理界線 | 不操弄、不做意識形態引導 |
| 辯證平衡 | 永遠同時 explore thesis 與 antithesis |
