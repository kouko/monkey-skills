# Aristotle's First Principles Skill

Decompose problems to fundamental truths and rebuild from scratch,
rejecting analogy and convention.

## The Five Phases

| Phase | Question | Purpose |
|-------|----------|---------|
| Problem Essence | What are you actually trying to achieve? | Separate outcomes from assumed solutions |
| Challenge Assumptions | What are you assuming and is it verified? | Test every assumption against evidence |
| Establish Ground Truths | What remains true no matter what? | Find 3-5 irreducible facts |
| Reason Upward | What's the simplest solution from only these truths? | Build minimal solution justified by truths |
| Validate Reasoning | Can every decision trace back to a ground truth? | Stress-test for analogy/complexity/legacy traps |

## Method Type

Process-driven (step-by-step decomposition and reconstruction).
Unlike Four Causes (analyze what exists) or Dialectics (compare positions).

## Key Traps to Avoid

| Trap | Description | Example |
|------|-------------|---------|
| Complexity | Unnecessary components sneak back in | Adding caching "just in case" |
| Analogy | Copying existing solutions unconsciously | "Like Uber for X" |
| Legacy | Maintaining obsolete compatibility | Supporting a format nobody uses |

## Examples in SKILL.md

| Example | Domain | Conventional Approach | First-Principles Result |
|---------|--------|----------------------|------------------------|
| Authentication | Internal tool | Add OAuth2 | Use existing corporate SSO |
| Data Storage | Event logging | PostgreSQL vs MongoDB | Append-only log files + batch query |
