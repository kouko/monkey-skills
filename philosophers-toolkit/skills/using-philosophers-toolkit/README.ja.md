# Using Philosophers Toolkit

[English](README.md) | **日本語** | [繁體中文](README.zh-TW.md)

ユーザーが適切な哲学的方法を見つけるのを助ける routing skill。

## 仕組み

ユーザーが何をしたいかを尋ね、最適な skill を推薦する：

| 意図 | 推奨される Skills |
|------|-------------------|
| 深く理解したい | Four Causes、First Principles、Shu-Ha-Ri |
| 決断したい | Dialectics、Falsifiability、Methodical Doubt、Ikigai |
| 改善したい | Kaizen、Wabi-Sabi |
| 振り返り・学びたい | Hansei、Socratic Method |
| 分からない | 絞り込みの問い → 既定で Socratic |

## Command

`/philosophers-toolkit:think`
