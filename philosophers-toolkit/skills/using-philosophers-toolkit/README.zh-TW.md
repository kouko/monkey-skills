# Using Philosophers Toolkit

[English](README.md) | [日本語](README.ja.md) | **繁體中文**

協助使用者找到合適哲學方法的 routing skill。

## 運作方式

詢問使用者想做什麼，再推薦最合適的 skill：

| 意圖 | 推薦的 Skills |
|------|---------------|
| 深入理解某事 | Four Causes、First Principles、Shu-Ha-Ri |
| 做決策 | Dialectics、Falsifiability、Methodical Doubt、Ikigai |
| 改善某事 | Kaizen、Wabi-Sabi |
| 反思或學習 | Hansei、Socratic Method |
| 不確定 | 收斂式提問 → 預設使用 Socratic |

## Command

`/philosophers-toolkit:think`
