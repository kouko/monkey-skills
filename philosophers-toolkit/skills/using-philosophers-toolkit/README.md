# Using Philosophers Toolkit

Routing skill that helps users find the right philosophical method.

## How It Works

Asks the user what they're trying to do, then recommends the best-fit skill:

| Intent | Recommended Skills |
|--------|-------------------|
| Understand something deeply | Four Causes, First Principles, Shu-Ha-Ri |
| Make a decision | Dialectics, Falsifiability, Methodical Doubt, Ikigai |
| Improve something | Kaizen, Wabi-Sabi |
| Reflect or learn | Hansei, Socratic Method |
| Not sure | Narrowing questions → default to Socratic |

## Command

`/philosophers-toolkit:think`
