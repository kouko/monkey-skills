# Scoreboard Builder

> Source: McChesney/Covey/Huling/Thele/Walker (2021), *The 4 Disciplines
> of Execution* 2nd ed., Simon & Schuster, p 250.

This is a fillable template — sketch your scoreboard directly into the
three blocks below, then validate it against the book's 8-question
self-check before declaring D3 done.

## Verbatim instruction (book p 250)

> Use this template to create a compelling scoreboard. Test your ideas
> against the checklist on the facing page.

## Validate before declaring done

Run the filled-in scoreboard through the book's official 8-item
self-check:

→ `../references/book-self-check.md` (McChesney et al. 2021, p 251 —
"Did You Get It Right?").

Any unchecked item is a return-to-bench signal — revise the block
above and re-test before treating the scoreboard as done.

## Scope note

For solo / personal scope, read "Team WIG" as "WIG" and "team" as
"you" throughout. The block structure (WIG + lag, lead-1 + graph,
lead-2 + graph) is unchanged. The 8-question gate has a personal-scope
lens in the same `book-self-check.md` file.
