# 4DX Meta — Team Strategy Triage

**English** | [日本語](README.ja.md) | [繁體中文](README.zh-TW.md)

> Pre-D1 gate for a team leader (3-12 reports) — answer "should THIS team adopt 4DX?" before any installation work begins.

## When this skill activates

- "Should our team adopt 4DX?" / "Will 4DX work for my team?"
- "Is 4DX right for a team of N?" / "Is my team ready for 4DX?"
- Leader has read the book / attended a workshop and is choosing operating cadence
- Team tried OKRs (or quarterly goals) and stalled — reaching for 4DX next
- Leader's boss / peer is pushing 4DX; leader wants a defensible yes/no/not-yet

## What it does (the protocol in brief)

The agent acts as a consultant to the leader, asking about the team (not the leader as an individual):

1. **Surface the team and the candidate goal** — team size + headline outcome in one sentence
2. **Stroke-of-the-pen check** — is there a single decision the leader controls that delivers the outcome? If yes → halt
3. **Whirlwind-handleable check** — could the team hit it just by doing existing work better? If yes → halt
4. **Team-shape screen** — size 3-12, not firefighting role, leader has time-authority, not single-shot project
5. **WIG-shape check** — *From X to Y by When* + mission-vision endorsement (P-45 veto)
6. **Adoption-curve readiness** — Models / Not Yets / Nevers split (F-28)
7. **Matched-set commitment** — all four disciplines or none (P-23)
8. **Verdict** — TEAM-APPLICABLE → proceed to D1, or one of 5 redirect verdicts

## When NOT to use

| Situation | Where to go instead |
|---|---|
| Solo / individual-contributor goal (no team) | `4dx-meta-personal-strategy-triage` |
| Team already committed; asking how to start D1 | `4dx-d1-team-primary-wig-selection` |
| Leader-of-Leaders multi-team rollout | Read book Ch 6-10 + use `4dx-d1-team-wig-cascade` |
| Specific-discipline question (lead measures, scoreboard) | Matching D-skill |
| Generic team-strategy question with no 4DX framing | Other decision skills |
