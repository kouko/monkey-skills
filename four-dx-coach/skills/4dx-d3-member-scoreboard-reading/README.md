# 4DX D3 — Member Scoreboard Reading

**English** | [日本語](README.ja.md) | [繁體中文](README.zh-TW.md)

> Team-member scope — read the team scoreboard your leader designed. Diagnose 5-second test, locate your contribution, spot calibration drift, escalate breakage non-confrontationally.

## When this skill activates

- "How do I read my team's scoreboard?"
- "Our team has a scoreboard — what should I be looking at?"
- "I look at our scoreboard and can't really tell what it's saying."
- "Where do I show up on the team scoreboard?"
- "Our scoreboard hasn't moved in weeks — is that real?"
- 「team の scoreboard をどう読む？」「我們 team 的 scoreboard 怎麼看」

## What it does (the protocol in brief)

≤5-minute weekly diagnostic — invert the leader's design rubric into a member's read rubric. Do NOT redesign.

1. **Comprehension gate** — locate the scoreboard medium / accessibility; halt to D3-team-design if no board exists
2. **5-second test (member-side)** — winning / losing / can't tell? "Can't tell" is data, not a member problem
3. **Locate-my-contribution per lead** — where does my work plug in? unable on ANY lead → D2 influence-mapping gap
4. **Trend read on focus leads** — week-over-week vs pacing line; does my activity plausibly explain the move?
5. **Calibration check** — has the pacing / target / definition drifted vs current reality?
6. **Decide: read complete or escalate** — if any flag fires (5-sec-fails / drift / hidden / stale), draft non-confrontational ask
7. **Output read card** — medium / 5-sec / contribution / trend / calibration / flags / escalation status

## When NOT to use

| Situation | Where to go instead |
|---|---|
| Designing a scoreboard (solo / leader) | `4dx-d3-personal-scoreboard` / `4dx-d3-team-lead-scoreboard-design` |
| No team scoreboard exists | Halt; route leader to `4dx-d3-team-lead-scoreboard-design` |
| Forward weekly commitment prep | `4dx-d4-member-commitment-prep` (consumes this read) |
| Backward debrief on a missed commitment | `4dx-d4-member-account-debrief` (consumes this read) |
| D2 influence-mapping gap (can't locate self on ANY lead) | `4dx-d2-member-lead-measure-influence` first |
| BI / KPI dashboards / executive scorecards | Out of 4DX scope |
