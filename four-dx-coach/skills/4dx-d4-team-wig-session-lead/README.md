# 4DX D4 — Team WIG Session (Leader Facilitation)

**English** | [日本語](README.ja.md) | [繁體中文](README.zh-TW.md)

> Coach you (the leader) to facilitate a 20-30 min weekly WIG Session — Account → Review → Plan — where members propose their own commitments and you veto-but-never-dictate.

## When this skill activates

- "Run my team's weekly WIG Session" / "Facilitate our team WIG meeting"
- D1-D3 installed (Team WIG + lead measures + scoreboard); first session pending
- WIG Session has drifted into a status meeting — no specific commitments, no movement
- A member missed a commitment and you don't know how to respond without shaming or eroding the standard
- You're travelling and tempted to skip the session

## What it does (the protocol in brief)

Consultant-to-leader. The agent whispers to you; you run the room:

1. **Pre-session check (≤2 min)** — scoreboard updated, last week's commitments visible, time-box set, WIG stated verbatim
2. **Frame the session (≤2 min)** — state WIG verbatim; name the three segments
3. **Segment 1 — Account, leader first then round-robin (~7-10 min)** — model reporting yourself; run the three-step on missed commitments (Respect → Accountability → Encourage)
4. **Segment 2 — Review the scoreboard (~5-7 min)** — probe trend, ask the team to interpret first, recognize wins by name
5. **Segment 3 — Plan, member proposes, leader gates (~8-10 min)** — apply the four-part specificity gate; veto-not-dictate; cap 1-2 per member
6. **Confirm out loud, then log (~2 min)** — every member states their final commitment verbatim
7. **Close + confirm next session (~1 min)** — same time, same place; if travelling, dial in or delegate chair, never skip

## When NOT to use

| Situation | Where to go instead |
|---|---|
| Solo user with no peer team | `4dx-d4-personal-wig-session` |
| Team member preparing their own commitment | `4dx-d4-member-commitment-prep` |
| Team member processing last week's outcome before the session | `4dx-d4-member-account-debrief` |
| Goal-discovery / WIG-selection conversation | A D1 team skill |
| Daily standup / scrum / 1:1 / quarterly review | Wrong artifact — WIG Session has 3 fixed segments |
