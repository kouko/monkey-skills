# 4DX D4 — Team WIG Session（Leader Facilitation）

[English](README.md) | **日本語** | [繁體中文](README.zh-TW.md)

> あなた（leader）を coach し、20-30 分の weekly WIG Session を進行する — Account → Review → Plan で、メンバーが自身の commitment を提案し、あなたは veto はするが dictate はしない。

## このスキルが起動するとき

- 「チームの WIG Session を運営する」「チーム WIG ミーティングのファシリテーションを助けて」
- D1-D3 導入済（Team WIG + lead measure + scoreboard）で初回 session 直前
- WIG Session が status meeting に流れ、specific commitment も進捗も出ない
- メンバーが commitment を果たせず、shame せず標準を緩めない応答が分からない
- 出張中で session を skip したくなっている

## 何をするか（プロトコル概要）

Leader への consultant。Agent はあなたに耳打ちし、部屋はあなたが回す:

1. **会前 check（≤2 min）** — scoreboard 更新、先週 commitment 可視、time-box、WIG を verbatim で
2. **開幕（≤2 min）** — WIG を verbatim で読み上げ、3 segment を宣言
3. **Segment 1 — Account（leader 先、後 round-robin、~7-10 min）** — まず自分が報告。kept failure 時は Respect → Accountability → Encourage の三段
4. **Segment 2 — Review scoreboard（~5-7 min）** — trend を見る、まずチームに解釈させる、win は名指しで認知
5. **Segment 3 — Plan（member 提案、leader が gate、~8-10 min）** — 4 部分の specificity gate、veto はするが dictate しない、1 人 1-2 で cap
6. **声に出して confirm + log（~2 min）** — 各員が最終 commitment を verbatim で再読
7. **closing + 次回確認（~1 min）** — same time same place、出張なら dial in / chair 委譲、skip 厳禁

## 使わない場面

| 状況 | 代わりの行き先 |
|---|---|
| solo user、peer チーム不在 | `4dx-d4-personal-wig-session` |
| メンバーが今週の commitment を準備 | `4dx-d4-member-commitment-prep` |
| メンバーが session 前に先週分を処理 | `4dx-d4-member-account-debrief` |
| まだ goal-discovery / WIG selection 段階 | D1 team skill |
| daily standup / sprint / 1:1 / 四半期 review | 別アーティファクト — WIG Session は 3 segment 固定 |
