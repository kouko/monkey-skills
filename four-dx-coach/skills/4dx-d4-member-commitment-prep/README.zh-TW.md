# 4DX D4 — Member Commitment Prep（會前備戰）

[English](README.md) | [日本語](README.ja.md) | **繁體中文**

> 帶你（團隊成員）在走進 team 的 WIG Session 前，5-10 分鐘內備好一條 specific、會推動 lead measure 的 commitment，過五道 filter。

## 何時觸發這個 skill

- 「我要為下次 WIG Session 準備 commitment」「下週的 commitment 我想先想清楚再去開會」
- 上次 commitment 太模糊或沒守住，懷疑問題出在 commitment 本身
- 不想在 peer / leader 面前出糗，想私下排演
- 一直把 whirlwind task 當 commitment（覺得急、但不會動 lead measure）

## 它做什麼（protocol 摘要）

私下備戰，personal-coach 對你發聲。Agent 可以打槍爛 commitment — 但不替你決定：

1. **Gate — 先確認 lead measure 已知（≤1 min）** — 不知道 → halt，去 comprehension skill
2. **回看上週 commitment（~1-2 min）** — kept / partial / not kept；沒守住給一個 root cause
3. **產出 candidate commitments（~2-3 min）** — 經典提問：「這週 ONE 件最重要、能動 lead measure、在你自己控制內的事」
4. **過五道 filter（~3-4 min，核心）** — Specific / 動 lead measure / 在自己控制內 / 自己選的 / cap 1-2 件
5. **排演 wording（~1 min）** — 一字不差說一次；偷溜回模糊話術 → 校正一次
6. **送你進房間（≤1 min）** — 進去後別在社交壓力下軟化它

## 不要在這些情境使用

| 情境 | 改去 |
|---|---|
| solo 用戶、沒有 team WIG Session | `4dx-d4-personal-wig-session` |
| Session 之「後」要 review 上週 | `4dx-d4-member-account-debrief` |
| 不知道團隊 lead measure | 先 `4dx-d1-member-team-wig-comprehension` |
| 你是主持會議的 leader | `4dx-d4-team-wig-session-lead` |
| daily standup / sprint task / OKR check-in commitment | 不同產物 — 不要混 |
