# 4DX D4 — Member Commitment Prep（Walk-In Ready）

[English](README.md) | **日本語** | [繁體中文](README.zh-TW.md)

> あなた（チームメンバー）を coach し、team の WIG Session に入る前に SPECIFIC で lead measure を動かす commitment を 5-10 分で準備する。5 つの filter で研ぎ澄ませる。

## このスキルが起動するとき

- 「次の WIG Session の commitment を準備したい」
- 「来週のミーティングで何をコミットすべきか相談したい」
- 前回 commitment が曖昧 / 守れなかった、commitment 自体が問題と疑っている
- peer / leader の前で恥をかきたくない、private に rehearsal したい
- whirlwind task を commitment にしている（urgent だが lead を動かさない）

## 何をするか（プロトコル概要）

Private prep、personal-coach 声であなたへ。Agent は悪い commitment に push back する — dictate しない:

1. **Gate — lead measure が既知か確認（≤1 min）** — 言えなければ halt、comprehension skill へ
2. **先週の commitment を振り返る（~1-2 min）** — kept / partial / not kept、守れなかった root cause を 1 つ
3. **candidate commitments を生成（~2-3 min）** — 正典的問い:「team の lead measure を動かす、自分の制御内で、今週 ONE 件」
4. **5 つの filter を適用（~3-4 min、核心）** — Specific / lead-measure-moving / 自制御内 / 自選 / scope-cap 1-2
5. **wording を rehearsal（~1 min）** — verbatim で口に出す、曖昧化したら 1 回修正
6. **session へ送り出す（≤1 min）** — 中で社交圧で軟化させない

## 使わない場面

| 状況 | 代わりの行き先 |
|---|---|
| solo user、team WIG Session なし | `4dx-d4-personal-wig-session` |
| session 「後」の先週 review | `4dx-d4-member-account-debrief` |
| team の lead measure を知らない | まず `4dx-d1-member-team-wig-comprehension` |
| あなたが session を回す leader | `4dx-d4-team-wig-session-lead` |
| daily standup / sprint task / OKR check-in commitment | 別アーティファクト — 混同しない |
