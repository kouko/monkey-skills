# 4DX D4 — Member Account Debrief (Walk Into the Session Clean)

**English** | [日本語](README.ja.md) | [繁體中文](README.zh-TW.md)

> Coach you (the team member) through an honest self-account of last week's commitment BEFORE the WIG Session — and prepare the ~30-second statement you will deliver in the Account segment.

## When this skill activates

- "I missed my commitment last week — how do I show up at the session?"
- "WIG Session is tomorrow and I didn't finish. What do I say?"
- "I half-did it. Is that a kept or a miss?"
- "I shipped Friday instead of Thursday with a regression — kept or not?"
- You're dreading the session and tempted to no-mention or excuse

## What it does (the protocol in brief)

Personal-coach to you, locker-room brief, ~5 minutes. Output: a 30-second Account script — no shame, no excuse, just straight:

1. **Open & set the frame (≤30 sec)** — script for the session, not therapy
2. **Q1 — Restate the commitment verbatim** — exact words. If you can't recall → that's data (P-22 violation upstream)
3. **Q2 — Classify the outcome straight** — kept / partial / missed in one word. P-22 two-part test: specific deliverable + on the committed window
4. **Q3 — If missed/partial: what blocked it?** — factual, not blame; reclaim agency from excuse-mode; name the Marcus pattern if you took on someone else's urgent
5. **Q4 — What did I learn?** — about commitment design / week capacity / lead-measure mechanism (Crisafulli discipline applies to wins too)
6. **Q5 — Re-commit or redesign?** — 1 miss → re-commit same / 2 misses → resize down / 3+ → redesign (route to commitment-prep)
7. **Compose the 30-second script** — commitment → outcome → cause → learning → next; checked against five failure modes (excuse / shame / vague / no-mention / over-commit)
8. **Close** — deliver it, sit down. Plan segment is where you'll re-commit, not the Account segment

## When NOT to use

| Situation | Where to go instead |
|---|---|
| Solo user with no team session | `4dx-sustain-personal-momentum-rescue` |
| 3+ misses in a row (commitment-design problem, not debrief) | `4dx-d4-member-commitment-prep` |
| Picking the next commitment | `4dx-d4-member-commitment-prep` |
| Leader-side facilitation / receiving the account | `4dx-d4-team-wig-session-lead` |
| Therapy / shame work — "I'm a failure" register | Upstream of 4DX; out of scope |
