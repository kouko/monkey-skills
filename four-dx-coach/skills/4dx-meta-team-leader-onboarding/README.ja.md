# 4DX Meta — Team Leader Onboarding

[English](README.md) | **日本語** | [繁體中文](README.zh-TW.md)

> あなた（leader-of-leaders）を coach し、直属 team-leader から compliance ではなく genuine commitment を引き出す。**2 mode**: **coach-mode**（立ち上げ前の buy-in 対話準備）と **audit-mode**（過去の onboarding が compliance / 反発を生んだ場合、rollout メール + 1:1 議事録 + チームフィードバックから診断）。

## このスキルが起動するとき

**coach-mode**（準備フェーズ — 過去の onboarding なし）:
- 「部下リーダーに 4DX を腹落ちさせたい」
- 「直属マネージャーが反発しそう、どう導入する」
- 「また流行りの施策と思われたくない」
- 「全社展開かパイロットか」「〇〇さんが反対しそう、どう話す」
- 以前 OKR / scorecards が leader 層で失速した経験があり、繰り返したくない

**audit-mode**（診断フェーズ — 過去 onboarding の artifact あり）:
- 「チーム導入したが反発が強い、何が間違ってた？」
- 「リーダー陣が表面的に従ってるだけ、何が抜けた？」
- 「ロールアウトメールと 1:1 議事録、見て診断して」

## 何をするか（プロトコル概要）

**coach-mode** — 単一の leader-of-leaders に対する consultant。成果は per-leader plan + リハーサル対話 1 本（rollout 文書ではない）:

1. **前提確認** — 4DX 適合済 + Primary WIG / Key Battles のドラフト保有
2. **leaders を名指し + skepticism レベル付け** — 3-5 名（enthusiast / neutral / skeptic / hard-no）
3. **各 leader: 彼らは何を望んでいる？** — あなたの声ではなく「彼らの」声で
4. **各 leader の最強反対意見を steel-man** — ラベルではなく完全文。反論はまだしない
5. **Pilot vs hold-back 分類** — voluntary pilot / hold back / engage but don't push
6. **ONE leader 用に対話をリハーサル** — 最高 leverage の相手、user 自身の言葉で口に出す
7. **per-leader plan + リハーサル script を出力** — 今週から動ける

**audit-mode** — consultant-from-artifacts。rollout メール / kickoff スライド / 1:1 議事録 / チームフィードバックを読み、4 lens（commitment-vs-compliance · Kotter Stage-1 urgency · per-leader steel-manning · pilot-vs-hold-back）で診断、critique を違反 rule にマッピング、per-leader recovery plan + 最高 leverage の missed leader 向け会話 revision + onward route を出力。

## 使わない場面

| 状況 | 代わりの行き先 |
|---|---|
| 組織への 4DX 適合判断が未了 | まず `4dx-meta-strategy-triage` |
| Buy-in 確保済、技術 cascade が次 | `4dx-d1-wig-cascade` |
| frontline メンバーの buy-in | D1 Team WIG 選択プロセス、Ch 8 ではない |
| 4DX 非依存の一般 change management | Kotter / ADKAR / Bridges |
| onboarding 後に cadence が連続スキップで崩壊 | `4dx-sustain-momentum-rescue` |
| Cross-layer audit（WIG + leads + scoreboard + cadence） | `4dx-audit` |
| 実は commitment ではなく compliance が欲しい | 停止 — この skill では助けにならない |
