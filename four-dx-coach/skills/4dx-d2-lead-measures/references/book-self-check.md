# D2 lead-measure book self-check (verbatim)

> Source: McChesney/Covey/Huling/Thele/Walker (2021), *The 4 Disciplines
> of Execution* 2nd ed., Simon & Schuster, p 229 (Ch 13, "Step 3: Test
> Top Ideas") + p 235 (form requirements).

This is the book's official "Did you get it right?" gate for any
candidate lead measure. Run every surviving candidate through all
seven items before adopting it. A FAIL on any single item disqualifies
the candidate; revise or replace.

## The 6 test criteria (book p 229, verbatim)

- [ ] 1. Is the measure predictive of achieving the Team WIG?
- [ ] 2. Can the team influence the measure?
- [ ] 3. Is it an ongoing process or a "one-and-done"?
- [ ] 4. Is it a leader's game or a team game?
- [ ] 5. Can it be measured?
- [ ] 6. Is it worth measuring?

## The +1 form requirement (book p 235, verbatim)

- [ ] 7. Does it start with a verb? Is it simple?

## Personal-scope lens (note, not a rewrite)

When the skill is run in personal scope, read the verbatim items above
through this lens — the source wording above is preserved unchanged:

- "Team WIG" → your WIG
- "the team" → you
- "leader's game or a team game" → "a delegated / done-for-you game,
  or a game you yourself play this week"

The verbatim version is the gate of record; the personal lens is for
applying the gate, not replacing it.

## How to use

1. List your candidate lead measures (typically 2-3 after the Ch 13
   ranking step).
2. For each candidate, walk all 7 items in order. Record PASS / FAIL +
   one-line reason per item.
3. Any FAIL → revise the candidate or drop it. Re-run.
4. Only candidates that PASS all 7 items proceed to D3 (scoreboard).
