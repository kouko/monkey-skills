# 4DX Meta-Strategy Triage

[English](README.md) | **日本語** | [繁體中文](README.zh-TW.md)

> 4 つの discipline をインストールする *前* に、その目標に 4DX が本当に合うかを判定する。

## このスキルが起動するとき

- 「この目標に 4DX 使える？」「4DX 適してる？」
- 4DX を読んだ・聞いたばかりで、自分の状況に当てはまるか確かめたい
- OKR / GTD / atomic habits / 気合で失敗してきて、次の手段として 4DX に手を伸ばしている
- 習慣形成・creative project・on-call role に 4DX を当てようとしている疑念がある
- D1 前のゲート — personal-coach パイプラインの最上流で WIG-defining より先に動く

## 何をするか（プロトコル概要）

Socratic 7 ステップで、6 種類の verdict のいずれかを返す:

1. **goal を具体化する** — 分類できる粒度で明示してもらう
2. **stroke-of-the-pen check** — 単一の判断・購入・採用で済むなら停止
3. **whirlwind-handleable check** — 既存の routine を少し磨くだけで足りるなら停止
4. **個人スケール anti-pattern screen** — habit / portfolio-bet / emergency role / creative output
5. **time-sovereignty check** — 週時間の ~20% を自分でコントロールできているか確認
6. **matched-set capacity check** — 4 つ全部やるか・やらないかの二択（つまみ食い不可）
7. **verdict + handoff** — `APPLICABLE` なら D1 へ、それ以外は redirect verdict を返す

返り得る verdict: `APPLICABLE` / `stroke-of-the-pen` / `whirlwind-handleable` / `habit-formation` / `portfolio-bet` / `emergency-role` / `creative-output` / `no-time-sovereignty`。

## 使わない場面

| 状況 | 代わりの行き先 |
|---|---|
| すでに 4DX に commit していて「どう始める？」と聞いている | `4dx-d1-personal-whirlwind-triage` → `4dx-d1-personal-wig-defining` |
| 特定の discipline について聞いている | 該当する D スキル |
| enterprise 展開 / 複数チームへの cascading WIG | 書籍 6-10 章（Leader-of-Leaders）を直接参照 |
| 4DX 語彙のない汎用的な意思決定 | 他の意思決定系スキル |
