# 4DX Meta-Strategy Triage

**English** | [日本語](README.ja.md) | [繁體中文](README.zh-TW.md)

> Decide whether 4DX even fits your goal — *before* installing any of the four disciplines.

## When this skill activates

- "Should I use 4DX for [X]?" / "Is 4DX right for my goal?"
- Just finished reading or hearing about 4DX and asking whether it applies
- Tried OKRs / GTD / atomic habits / willpower and now reaching for 4DX as next-thing-to-try
- Suspect 4DX is being over-applied to a habit, a creative project, or an on-call role
- Pre-D1 entry point — gates the personal-coach pipeline before any WIG-defining work begins

## What it does (the protocol in brief)

A Socratic 7-step triage that returns one of 6 verdicts:

1. **Surface the goal in concrete terms** — user names the goal explicitly enough to classify
2. **Stroke-of-the-pen check** — if a single decision/purchase/hire suffices, halt with verdict
3. **Whirlwind-handleable check** — if existing routine sharpened a bit suffices, halt
4. **Personal anti-pattern screen** — habit / portfolio-bet / emergency role / creative output
5. **Time-sovereignty check** — confirm at least ~20% of weekly hours are user-controlled
6. **Matched-set capacity check** — user commits to all four disciplines or none (no cherry-pick)
7. **Verdict + handoff** — `APPLICABLE` proceeds to D1; otherwise issues a redirect verdict

Possible verdicts: `APPLICABLE` / `stroke-of-the-pen` / `whirlwind-handleable` / `habit-formation` / `portfolio-bet` / `emergency-role` / `creative-output` / `no-time-sovereignty`.

## When NOT to use

| Situation | Where to go instead |
|---|---|
| User already committed to 4DX, asks "how do I start?" | `4dx-d1-personal-whirlwind-triage` then `4dx-d1-personal-wig-defining` |
| User asks about a specific discipline | The matching D-skill |
| Enterprise rollout / cascading WIGs across teams | Read book Chapters 6-10 (Leader-of-Leaders) directly |
| Generic decision-gate question with no 4DX vocabulary | Other decision-method skills |
